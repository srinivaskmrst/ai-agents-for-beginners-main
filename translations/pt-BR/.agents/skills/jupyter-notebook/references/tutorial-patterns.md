# Padrões de Tutorial

Use esta estrutura para ensino e tutoriais guiados:

- Público, pré-requisitos e objetivos de aprendizagem: diga para quem é, liste o que eles já deveriam saber e indique o que serão capazes de fazer ao final.
- Esboço: forneça um breve sumário numerado para que os leitores possam consultar rapidamente.
- Fluxo passo a passo: combine uma breve explicação em markdown com uma pequena célula de código que seja executada por conta própria e uma breve interpretação do resultado.
- Exercícios: inclua pelo menos um exercício que reforce o conceito-chave e forneça um esqueleto de resposta na próxima célula.
- Armadilhas e extensões: destaque um erro comum e como corrigi-lo, e sugira uma extensão opcional para leitores curiosos.
