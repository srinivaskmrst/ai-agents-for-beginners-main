> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```

## 🌱 شروع کریں

یہ کورس AI ایجنٹس بنانے کے بنیادی اصولوں پر مبنی دروس پر مشتمل ہے۔ ہر سبق اپنے موضوع کو کور کرتا ہے، لہٰذا جہاں چاہیں شروع کریں!

اس کورس کے لیے کثیر الزبان تعاون موجود ہے۔ ہماری [دستیاب زبانیں یہاں دیکھیں](#-multi-language-support)۔ 

اگر یہ آپ کا پہلا موقع ہے جب آپ Generative AI ماڈلز کے ساتھ کام کر رہے ہیں، تو ہمارے [ابتدائیوں کے لیے جنریٹو AI](https://aka.ms/genai-beginners) کورس کو دیکھیں، جس میں جنریٹو AI کے ساتھ کام کرنے کے لیے 21 دروس شامل ہیں۔

اس ریپو کو [ستارہ (🌟) دینا](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) اور [اس کا فورک بنانا](https://github.com/microsoft/ai-agents-for-beginners/fork) نہ بھولیں تاکہ آپ کوڈ چلا سکیں۔

### دوسرے سیکھنے والوں سے ملیں، اپنے سوالات کا جواب پائیں

اگر آپ کو کہیں پھنسا ہوا محسوس ہو یا AI ایجنٹس بنانے کے متعلق سوالات ہوں تو ہمارے مخصوص Discord چینل میں شامل ہوں جو [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) میں موجود ہے۔

### آپ کو کیا چیز چاہیے

اس کورس کا ہر سبق کوڈ کی مثالوں کے ساتھ آتا ہے، جو کوڈ_samples فولڈر میں دستیاب ہیں۔ آپ اپنی کاپی بنانے کے لیے [اس ریپو کو فورک کر سکتے ہیں](https://github.com/microsoft/ai-agents-for-beginners/fork)۔  

یہ مشقوں میں کوڈ کی مثالیں Microsoft Agent Framework کے ساتھ Azure AI Foundry Agent Service V2 استعمال کرتی ہیں:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure اکاؤنٹ ضروری ہے

یہ کورس Microsoft کی مندرجہ ذیل AI Agent فریم ورکس اور خدمات استعمال کرتا ہے:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)  
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)  

کچھ کوڈ نمونے OpenAI-مطابق متبادل فراہم کنندگان جیسے [MiniMax](https://platform.minimaxi.com/) کی بھی حمایت کرتے ہیں، جو بڑے کانٹیکسٹ ماڈلز (204K ٹوکن تک) فراہم کرتا ہے۔ کنفیگریشن کی تفصیلات کے لیے دیکھیں [کورس سیٹ اپ](./00-course-setup/README.md)۔

اس کورس کے لیے کوڈ چلانے کی مزید معلومات کے لیے، جانیں [کورس سیٹ اپ](./00-course-setup/README.md)۔
