## 🌱 入门指南

本课程包含构建 AI 代理的基础课程。每节课覆盖独立主题，你可以从任何感兴趣的地方开始！

本课程支持多语言。请访问我们的[可用语言列表](#-multi-language-support)。

如果你是首次构建生成式 AI 模型，推荐查看我们的[生成式 AI 入门课程](https://aka.ms/genai-beginners)，该课程包含 21 节关于生成式 AI 构建的课程。

不要忘了给本仓库[点星 (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)并[Fork 本仓库](https://github.com/microsoft/ai-agents-for-beginners/fork)以便能运行代码。

### 认识其他学习者，获取问题解答

如果遇到难题或有关于构建 AI 代理的任何疑问，欢迎加入我们专门的 Discord 频道——[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)。

### 所需准备

本课程每节课都包含代码示例，位于 code_samples 文件夹中。你可以[Fork 本仓库](https://github.com/microsoft/ai-agents-for-beginners/fork)创建自己的副本。

代码示例使用 Microsoft Agent Framework 和 Azure AI Foundry Agent Service V2：

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要 Azure 账户

本课程使用微软提供的以下 AI 代理框架和服务：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

部分代码示例还支持如 [MiniMax](https://platform.minimaxi.com/) 等兼容 OpenAI 的替代服务，提供大上下文模型（最高支持 204K 令牌）。详细配置请参阅[课程设置](./00-course-setup/README.md)。

想了解本课程代码的运行详情，请访问[课程设置](./00-course-setup/README.md)。
