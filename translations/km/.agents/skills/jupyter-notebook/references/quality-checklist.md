# បញ្ជីពិនិត្យគុណភាព

Before delivering a notebook:

- រត់វាចាប់ពីលើទៅក្រោមយ៉ាងហោចណាស់ម្តង (ឬតាមដែលបរិយាកាសអនុញ្ញាត).
- ប្រាកដថាកោសិកាដើមទាំងអស់កំណត់ស្ថានភាពដែលត្រូវការ; ជៀសវាងស្ថានភាពលាក់ពីការរត់មុន។
- រក្សាលទ្ធផលឲ្យមានរបៀបស្អាត។ ជៀសវាងលទ្ធផលធំនោះនៅពេលសេចក្តីសង្ខេបខ្លីគ្រប់គ្រាន់។
- ជ្រើសរើសតារាងតូចៗ, មាត្រដ្ឋានសំខាន់ៗ, ឬការបោះពុម្ពខ្លីៗ។
- រៀបរយអត្ថាធិប្បាយឲ្យអាចអានឆាប់ៗ។ ប្រើចំណងជើង និងចំណុចខ្លីៗ ហើយជៀសវាងកថាខណ្ឌវែង។
- ទុក TODOs ដែលមានប្រយោជន៍តែពេលចាំបាច់ ហើយសម្គាល់ពួកវាឱ្យច្បាស់។
- ប្រសិនបើមិនអាចប្រតិបត្តិបាន សូមបញ្ជាក់ពីហានិភ័យ និងវិធីសម្រាប់ផ្ទៀងផ្ទាត់នៅលើម៉ាស៊ីនរបស់អ្នក។
