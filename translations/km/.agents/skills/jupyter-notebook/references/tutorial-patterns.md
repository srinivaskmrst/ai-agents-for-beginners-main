# លំនាំសម្រាប់មេរៀន

ប្រើរចនាសម្ព័ន្ធនេះសម្រាប់ការបង្រៀន និងការដឹកនាំដោយជំហាន៖

- អ្នកទស្សនា, លក្ខខណ្ឌមុន, និងគោលដៅការសិក្សា: បញ្ជាក់ថាសម្រាប់នរណា, បញ្ជីអ្វីដែលពួកគេគួរតែបានដឹងរួច, និងបញ្ជាក់អំពីអ្វីដែលពួកគេនឹងអាចធ្វើបាននៅចុងបញ្ចប់។
- អារាយដ្ឋាន: ផ្តល់អារាយដ្ឋានខ្លីដែលមានលេខ ដើម្បីឱ្យអ្នកអានអាចមើលឆាប់ៗ។
- ជំហាន-ដោយ-ជំហាន: លាយការពិពណ៌នាសង្ខេបក្នុង Markdown ជាមួយកោដសែលតូចមួយដែលអាចដំណើរការបានដោយខ្លួនឯង ហើយផ្តល់ការបកស្រាយខ្លីអំពីលទ្ធផល។
- លំហាត់: រួមបញ្ចូលយ៉ាងតិចមួយលំហាត់ដែលពង្រឹងគំនិតសំខាន់ និងផ្តល់គ្រោងចម្លើយនៅក្នុងសែលបន្ទាប់។
- កំហុសដែលច្រើនប្រឈម និងការពង្រីក: បង្ហាញពីកំហុសទូទៅមួយ និងរបៀបជួសជុលវា ផងដែរ សូមស្នើសុំនូវការពង្រីកជាជម្រើសមួយសម្រាប់អ្នកអានដែលចង់ស្វែងយល់បន្ថែម។
