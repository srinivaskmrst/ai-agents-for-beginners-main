# бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһё Microsoft Agent Framework

![Agent Framework](../../../translated_images/km/lesson-14-thumbnail.90df0065b9d234ee.webp)

### бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

- бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһё Microsoft Agent Framework: бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— бһ“бһ·бһ„бһҸбҹҶбһӣбҹғ  
- бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ–бһёбһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣбһ“бҹғ Microsoft Agent Framework
- бһӣбҹҶбһ“бһ¶бһү MAF бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ: Workflows, Middleware, бһ“бһ·бһ„ Memory

## бһӮбҹ„бһӣбһҠбҹ…бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ–

- бһҖбһҹбһ¶бһ„ AI Agents бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Agent Framework
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ“бҹғ Microsoft Agent Framework бһ‘бҹ…бһӣбһҫбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Agentic бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ
- бһ”бҹ’бһҡбһҫбһӣбҹҶбһ“бһ¶бһүбһҮбҹҶбһ“бһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ бһҡбһҪбһҳбһҳбһ¶бһ“ workflows, middleware, бһ“бһ·бһ„ observability

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠ

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok) бһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҖбҹ’бһҡбҹ„бһҳбһҜбһҖбһҹбһ¶бһҡ `xx-python-agent-framework` бһ“бһ·бһ„ `xx-dotnet-agent-framework`бҹ”

## бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһё Microsoft Agent Framework

![Framework Intro](../../../translated_images/km/framework-intro.077af16617cf130c.webp)

[Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok) бһӮбһәбһҮбһ¶бһӮбҹ’бһҡбҹ„бһ„бһ”бһҺбҹ’бһҠбһ»бҹҶбһҸбҹӮбһҳбһҪбһҷбһҡбһ”бһҹбҹӢ Microsoft бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢ AI agentsбҹ” бһңбһ¶бһ”бҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһҡбһёбһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ agentic бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹӮбһӣбһ”бһ¶бһ“бһғбһҫбһүбһ‘бһ¶бҹҶбһ„бһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ Agent бһҠбҹ„бһҷбһӣбҹҶбһҠбһ¶бһ”бҹӢ** бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбҹҶбһўбһ·бһҸбһҮбҹҶбһ бһ¶бһ“вҖӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ workflows бһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бҹ—бҹ”
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһҹбһҳбҹҗбһҷбһҹбҹ’бһҡбһ”** бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбҹҶбһўбһ·бһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһІбҹ’бһҷ agents бһ”бһүбҹ’бһ…бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бҹ…бһ–бҹҒбһӣбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ”
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһҮбһҮбҹӮбһҖ** бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбҹҶбһўбһ·бһҸ бһҠбҹӮбһӣ agents бһўбһ¶бһ…бһҹбһ бһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһӣбһҫбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҸбҹӮбһҳбһҪбһҷбҹ”
- **бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…** бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбҹҶбһўбһ·бһҸбһҠбҹӮбһӣ agents бһ•бҹ’бһ‘бҹҒбһҡбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бҹ…бһӮбҹ’бһ“бһ¶ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҡбһ„бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ”бҹӢбҹ”
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһү** бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбҹҶбһўбһ·бһҸ бһҠбҹӮбһӣ agent бһ‘бһёбһ”бҹ’бһҡбһ№бһҖбҹ’бһҹбһ¶бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһҮбһёбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ бһҫбһҷбһҮбһҪбһҷбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ subagents бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһҮбһјбһ“ AI Agents бһҖбҹ’бһ“бһ»бһ„бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ MAF бһҖбҹҸбһ”бһ¶бһ“бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбҹ–

- **бһҖбһ¶бһҡбһҹбһ„бҹ’бһҖбҹҒбһҸбһҳбһҫбһӣ** бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ OpenTelemetry бһҠбҹӮбһӣбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ AI Agent бһҡбһ¶бһӣбҹӢбһҷбҹүбһ¶бһ„ бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһҮбҹҶбһ“бһҪбһүбһ…бһӣбһ“бһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҸбһ¶бһҳ Microsoft Foundry dashboardsбҹ”
- **бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–** бһҠбҹ„бһҷбһ•бһ№бһҖ agents бһҠбҹ„бһҷбһҠбһҫбһҡбһЎбһҫбһ„бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһӣбһҫ Microsoft Foundry бһҠбҹӮбһӣбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбһҪбһ“бһ¶бһ‘бһё бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҡбһҪбһ…бһ бһҫбһҷбҹ”
- **бһ—бһ¶бһ–бһ’бһ“бҹӢбһ‘бҹ’бһҡбһ¶бҹҶ** бһҠбҹ„бһҷбһҗбҹ’бһҡбһ„ Agent threads бһ“бһ·бһ„ workflows бһўбһ¶бһ…бһ”бһүбҹ’бһҲбһ”бҹӢ бһ”бһ“бҹ’бһҸ бһ“бһ·бһ„бһҹбҹ’бһҠбһ¶бһҡбһЎбһҫбһ„бһңбһ·бһүбһ–бһёбһҖбҹҶбһ бһ»бһҹ бһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡбһҮбһ¶бһ„бһҳбһ»бһ“бҹ”
- **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҖбһ¶бһҡбһҺбҹҚ** бһҠбҹ„бһҷбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ…бҹҶбһ бҹҠбһ»бһҳбһҳбһ“бһ»бһҹбҹ’бһҹбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҠбҹ’бһӢбһ¶бһ“бһҷбҹ„бһ„ бһҠбҹӮбһӣбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҗбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ“бһ»бһҳбҹҗбһҸбһ–бһёбһҳбһ“бһ»бһҹбҹ’бһҹбҹ”

Microsoft Agent Framework бһҖбҹҸбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ—бһ¶бһ–бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһҠбҹ„бһҷбҹ–

- **бһҳбһ·бһ“бҹӢбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ Cloud бһҺбһ¶бһҳбһҪбһҷбһ‘бҹҒ** - Agents бһўбһ¶бһ…бһҡбһҸбҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ containers, бһ“бҹ…бһӣбһҫ premises бһ“бһ·бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ cloud бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ…бҹ’бһҡбһҫбһ“бҹ”
- **бһҳбһ·бһ“бҹӢбһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ Provider бһ‘бҹҒ** - Agents бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҳ SDK бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ Azure OpenAI бһ“бһ·бһ„ OpenAI
- **бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһ‘бһ„бҹӢбһңбһ·бһ’бһ¶бһ“ Open Standards** - Agents бһўбһ¶бһ…бһ”бҹ’бһҡбһҫ protocols бһҠбһјбһ…бһҮбһ¶ Agent-to-Agent(A2A) бһ“бһ·бһ„ Model Context Protocol (MCP) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ agents бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”
- **Plugins бһ“бһ·бһ„ Connectors** - бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„ memory бһҠбһјбһ… Microsoft Fabric, SharePoint, Pinecone бһ“бһ·бһ„ Qdrantбҹ”

бһҳбһҖбһҳбһҫбһӣбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһӣбһҫбһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣбһҒбҹ’бһӣбҹҮбҹ—бһ“бҹғ Microsoft Agent Frameworkбҹ”

**бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· Agents**

Agents бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһё `.run` бһ¬ `.run_stream` бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бһ¶бҹҶбһ„бһҳбһ·бһ“бһ”бҹ’бһҡбһҫ stream бһ“бһ·бһ„бһ”бҹ’бһҡбһҫ streamбҹ”

```python
result = await agent.run("What are good places to visit in Amsterdam?")
print(result.text)
```
  
```python
async for update in agent.run_stream("What are the good places to visit in Amsterdam?"):
    if update.text:
        print(update.text, end="", flush=True)

```
  
бһҡбһ¶бһӣбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· agent бһўбһ¶бһ…бһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бһҸбһ¶бһҳбһ”бһҳбҹ’бһҡбҹӮбһ”бһҳбҹ’бһҡбһҪбһӣбһҠбһјбһ…бһҮбһ¶ `max_tokens` бһҠбҹӮбһӣ agent бһ”бҹ’бһҡбһҫ, `tools` бһҠбҹӮбһӣ agent бһўбһ¶бһ…бһ бҹ…, бһ“бһ·бһ„ бһ‘бҹ„бҹҮбһҮбһ¶бһ…бҹҶбһӣбһҫбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ `model` бһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҠбһјбһ…бһҠбҹӮбһӣбһҠбҹӮбһӣбһӮбҹҒбһ‘бһ¶бһҳбһ‘бһ¶бһҡбҹ”

бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ model бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”

**бһ§бһ”бһҖбһҡбһҺбҹҚ**

бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹ…бһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢ agentбҹ–

```python
def get_attractions( location: Annotated[str, Field(description="The location to get the top tourist attractions for")], ) -> str: """Get the top tourist attractions for a given location.""" return f"The top attractions for {location} are." 


# бһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ ChatAgent бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ

agent = ChatAgent( chat_client=OpenAIChatClient(), instructions="You are a helpful assistant", tools=[get_attractions]

```
  
бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһ–бҹҒбһӣбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸ agent бһ“бҹ„бҹҮбһ•бһ„бһҠбҹӮбһҡбҹ–

```python

result1 = await agent.run( "What's the best place to visit in Seattle?", tools=[get_attractions] # бһ§бһ”бһҖбһҡбһҺбҹҚбһ•бҹ’бһҸбһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ“бҹҒбҹҮбһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһҳбҹ’бһҠбһ„бһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ )
```
  
**Agent Threads**

Agent Threads бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“бҹ” Threads бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹ„бһҷбҹ–

- бһ”бҹ’бһҡбһҫ `get_new_thread()` бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ”бһүбҹ’бһ…бһјбһӣ thread бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷ
- бһ”бһ„бҹ’бһҖбһҫбһҸ thread бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ agent бһ бһҫбһҷ thread бһ“бҹ„бҹҮбһҳбһ¶бһ“бһўбһ¶бһҷбһ»бһҡбһҷбҹҲбһ–бҹҒбһӣбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ thread бһӮбһјбһҠбһҳбһҫбһӣбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```python
# бһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһ…бһ„бҹ’бһҡбҹ’бһҖбһ¶бһ“бһҗбҹ’бһҳбһёбһҳбһҪбһҷбҹ”
thread = agent.get_new_thread() # бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҒбҹ’бһҹбҹӮбһ…бһ„бҹ’бһҡбҹ’бһҖбһ¶бһ“бһ“бҹ„бҹҮбҹ”
response = await agent.run("Hello, I am here to help you book travel. Where would you like to go?", thread=thread)

```
  
бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ serialize thread бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбҹ–

```python
# бһ”бһ„бҹ’бһҖбһҫбһҸбһҒбҹ’бһҹбҹӮбһҗбҹ’бһҳбһёбһҳбһҪбһҷбҹ”
thread = agent.get_new_thread() 

# бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ—бһ¶бһӮбһёбһҮбһ¶бһҳбһҪбһҷбһҒбҹ’бһҹбҹӮбҹ”

response = await agent.run("Hello, how are you?", thread=thread) 

# бһҡбҹҖбһ”бһ…бҹҶбһҒбҹ’бһҹбҹӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбҹ”

serialized_thread = await thread.serialize() 

# бһ”бҹҶбһ”бҹӮбһҖбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҒбҹ’бһҹбҹӮбһҖбҹ’бһҡбҹ„бһҷбһ–бһёбһ”бһҫбһҖбһ–бһёбһҹбҹ’бһҸбһ»бһҖбҹ”

resumed_thread = await agent.deserialize_thread(serialized_thread)
```
  
**Agent Middleware**

Agents бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„ LLM бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ” бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһӣбҹҶбһўбһ·бһҸбһҒбҹ’бһӣбҹҮбҹ— бһҷбһҫбһ„бһ…бһ„бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ¬бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„ agent бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбҹ” Agent middleware бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“бһҸбһ¶бһҳбһҡбһҷбҹҲбҹ–

*Function Middleware*

Middleware бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһңбһ¶бһ„ agent бһ“бһ·бһ„ function/бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһңбһ¶бһ“бһ№бһ„бһ бҹ…бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бһҫбһ…бһ„бҹӢбһ…бһ»бҹҮбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһ¶бһӣбһҺбһ¶бһ бҹ… function бһ“бҹ„бҹҮбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳ `next` бһҖбҹҶбһҺбһҸбҹӢбһҗбһ¶ middleware бһ¬ function бһ‘бһёбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бҹ”

```python
async def logging_function_middleware(
    context: FunctionInvocationContext,
    next: Callable[[FunctionInvocationContext], Awaitable[None]],
) -> None:
    """Function middleware that logs function execution."""
    # бһҖбһ¶бһҡбһ”бҹҶбһҺбһ¶бһҹбҹӢбһҳбһ»бһ“бҹ– бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ“бһ¶бһ–бҹҒбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·
    print(f"[Function] Calling {context.function.name}")

    # бһ”бһ“бҹ’бһҸбһ‘бҹ…бһҳбҹҒбһҢбҹҖбһңбҹҒбһҡбһҡбһәбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ
    await next(context)

    # бһҖбһ¶бһҡбһ”бҹҶбһҺбһ¶бһҹбҹӢбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ– бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·
    print(f"[Function] {context.function.name} completed")
```
  
*Chat Middleware*

Middleware бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ¬бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһңбһ¶бһ„ agent бһ“бһ·бһ„бһҹбҹҶбһҺбһҫбһҡбһңбһ…бһ“бһ¶бһ“бһ»бһҖбҹ’бһҡбһҳ (LLM)бҹ”

бһңбһ¶бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҗбһҸбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһҠбһјбһ…бһҮбһ¶ `messages` бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһүбһҫбһ‘бҹ…бһҹбҹҒбһңбһ¶ AIбҹ”

```python
async def logging_chat_middleware(
    context: ChatContext,
    next: Callable[[ChatContext], Awaitable[None]],
) -> None:
    """Chat middleware that logs AI interactions."""
    # бһҖбһ¶бһҡ_Pre-processing: бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҮбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҳбһ»бһ“бһҖбһ¶бһҡбһ бҹ… AI
    print(f"[Chat] Sending {len(context.messages)} messages to AI")

    # бһ”бһ“бҹ’бһҸбһ‘бҹ… middleware бһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ
    await next(context)

    # бһҖбһ¶бһҡ_Post-processing: бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ” AI
    print("[Chat] AI response received")

```

### Workflows

MAF бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһң workflows бһҠбҹӮбһӣбһҮбһ¶бһҮбҹҶбһ бһ¶бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһҪбһ…бһҡбһҪбһ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ бһҫбһҷбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ AI agents бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһҮбҹҶбһ бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһ“бҹ„бҹҮбҹ”

Workflows бһҳбһ¶бһ“бһӮбҹ’бһҡбһҝбһ„бһ•бҹ’бһҹбҹҶбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбҹӮбһӣбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ…бһҡбһ“бҹ’бһҸбһ”бһ¶бһ“бһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбҹ” Workflows бһ“бҹҒбҹҮбһҖбҹҸбһўбһ¶бһ…бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ **бһҖбһ¶бһҡвҖӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚ agent бһ…бҹ’бһҡбһҫбһ“** бһ“бһ·бһ„ **checkpointing** бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ– workflowбҹ”

бһӮбҹ’бһҡбһҝбһ„бһ•бҹ’бһҹбҹҶбһҹбҹ’бһ“бһјбһӣбһ“бҹғ workflow бһҳбһ¶бһ“бҹ–

**Edges**

Edges бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһ…бһҡбһ“бҹ’бһҸmessages бһҖбҹ’бһ“бһ»бһ„ workflowбҹ” Edges бһўбһ¶бһ…бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһјбһ…бһҮбһ¶бҹ–

*Direct Edges* - бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҪбһҷбһ‘бҹ…бһҳбһҪбһҷбһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҹбһ¶бһҳбһүбҹ’бһүбһҡбһңбһ¶бһ„ executorsбҹ–

```python
from agent_framework import WorkflowBuilder

builder = WorkflowBuilder()
builder.add_edge(source_executor, target_executor)
builder.set_start_executor(source_executor)
workflow = builder.build()
```
  
*Conditional Edges* - бһҹбһҖбһҳбҹ’бһҳбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҒбҹ’бһӣбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ–бҹҒбһүбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бҹ…бһ–бҹҒбһӣбһ”бһ“бҹ’бһ‘бһ”бҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡбһҳбһ·бһ“бһҳбһ¶бһ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбҹҖбһ”бһ…бҹҶ бһўбҹ’бһ“бһҖбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҺбҹӮбһ“бһ¶бҹҶбһҮбһҳбҹ’бһҡбһҫбһҹбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

*Switch-case Edges* - бһ•бҹ’бһүбһҫ messages бһ‘бҹ… executors бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҖбҹҶбһҺбһҸбҹӢбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ”бһҫбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһҡбһҪбһҳбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҒбҹ’бһ–бһҹбҹӢ бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ”бһҹбҹӢбһӮбһ¶бһҸбҹӢбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҸбһ¶бһҳ workflow бһ•бҹ’бһҹбҹҒбһ„бҹ”

*Fan-out Edges* - бһ•бҹ’бһүбһҫбһ”бһ¶бһ“бһҳбһҪбһҷ message бһ‘бҹ…бһӮбҹ„бһӣбһҠбҹ…бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”

*Fan-in Edges* - бһ”бҹ’бһҡбһҳбһјбһӣ messages бһ–бһё executors бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ бһҫбһҷбһ•бҹ’бһүбһҫбһ‘бҹ…бһӮбҹ„бһӣбһҠбҹ…бһҸбҹӮбһҳбһҪбһҷбҹ”

**Events**

бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһҹбһ„бҹ’бһҖбҹҒбһҸбһҳбһҫбһӣбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ”бҹ’бһҡбһҹбһҫбһҡбһ…бҹҶбһ–бҹ„бҹҮ workflows, MAF бһ•бҹ’бһҸбһӣбҹӢбһўбҹҠбһёбһңбҹүбһ„бҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- `WorkflowStartedEvent`  - бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ workflow бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳ
- `WorkflowOutputEvent` - workflow бһ•бһӣбһ·бһҸбһ…бҹҒбһүбһ…бһҳбҹ’бһӣбһҫбһҷ
- `WorkflowErrorEvent` - workflow бһҮбһҪбһ”бһҖбҹҶбһ бһ»бһҹ
- `ExecutorInvokeEvent`  - Executor бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- `ExecutorCompleteEvent`  - Executor бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- `RequestInfoEvent` - бһҳбһ¶бһ“бһ–бҹ„бһ„бһҹбҹҶбһҺбһҫ

## бһӣбҹҶбһ“бһ¶бһү MAF бһҮбҹҶбһ“бһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ

бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһӣбһҫбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһӮбҹҶбһ“бһ·бһҸбһҹбҹ’бһ“бһјбһӣбһ“бҹғ Microsoft Agent Frameworkбҹ” бһ“бҹ…бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҖбһҹбһ¶бһ„ agents бһҹбҹ’бһҳбһ»бһӮвҖӢбһҹбҹ’бһҳбһ¶бһүбһҮбһ¶бһ„бһҳбһ»бһ“ бһҳбһ¶бһ“бһӣбҹҶбһ“бһ¶бһүбһҮбҹҶбһ“бһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бҹ–

- **бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ Middleware**: бһҹбһ„бҹ’бһҹбҹҗбһҷбһҸбҹҶбһҺбһҡбһҒбҹ’бһҹбҹӮ chained middleware бһӮбһәбһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ middleware бһ•бҹ’бһҹбҹҒбһ„бҹ— (logging, auth, rate-limiting) бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ function бһ“бһ·бһ„ chat middleware бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһҳбҹ’бһўбһ·бһҸбһӣбһҫбһ…бһҡбһ·бһҸбһ”бһ»бһӮбҹ’бһӮбһӣ agentбҹ”
- **Checkpointing Workflow**: бһ”бҹ’бһҡбһҫ workflow events бһ“бһ·бһ„ serialization бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖ бһ“бһ·бһ„бһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҷбһјбһҡбҹ”
- **бһҮбһҳбҹ’бһҡбһҫбһҹбһ§бһ”бһҖбһҡбһҺбҹҚбһҢбһёбһҺбһ¶бһҳбһ·бһ…**: бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ RAG бһӣбһҫбһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ§бһ”бһҖбһҡбһҺбҹҚ MAF бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһӮбһҸбҹӢбһҖбҹ’бһ“бһ»бһ„бһҹбҹҶбһҺбһҪбһҡбҹ”
- **Multi-Agent Handoff**: бһ”бҹ’бһҡбһҫ workflow edges бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“ conditional бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҒбһҡбһҮбһ¶agent бһҡбҹҖбһ”бһ…бҹҶбһҮбһ¶бһҹбҹҠбһ»бһҳбһ‘бһјбһӣбһ¶бһҷбҹ”

## бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠ

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Microsoft Agent Framework бһўбһ¶бһ…бһҡбһҖбһ”бһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҖбҹ’бһҡбҹ„бһҳбһҜбһҖбһҹбһ¶бһҡ `xx-python-agent-framework` бһ“бһ·бһ„ `xx-dotnet-agent-framework`бҹ”

## бһҳбһ¶бһ“бһҹбҹҶбһҺбһҪбһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһўбҹҶбһ–бһё Microsoft Agent Framework?

бһ…бһјбһӣбһҡбһҪбһҳбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„В [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)В бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһ”бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһҡбҹҖбһ“бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһ…бһјбһӣбһҡбһҪбһҳбһҳбҹүбҹ„бһ„бһҖбһ¶бһҡбһ·бһҷбһ¶бһӣбҹҗбһҷ бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бһҳбҹ’бһӣбһҫбһҷбһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҶбһҺбһҪбһҡбһўбҹҶбһ–бһё AI Agents бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
