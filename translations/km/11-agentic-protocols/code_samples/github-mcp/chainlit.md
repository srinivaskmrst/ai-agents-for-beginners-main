# ស្វាគមន៍មកកាន់ Chainlit! 🚀🤖

សួស្តី អ្នកអភិវឌ្ឍន៍! 👋 យើងរំភើបដែលមានអ្នកចូលរួមជាមួយយើង។ Chainlit គឺជាឧបករណ៍មានសមត្ថភាព ដែលរចនាឡើងដើម្បីជួយអ្នកក្នុងការបង្កើតគំរូ (prototype), ដោះស្រាយកំហុស (debug) និងចែករំលែកកម្មវិធីដែលស្ថាបនាលើ LLMs។

## តំណរប្រើប្រាស់ដែលមានប្រយោជន៍ 🔗

- **Documentation:** ចាប់ផ្តើមជាមួយ [ឯកសារ Chainlit](https://docs.chainlit.io) របស់យើង 📚
- **Discord Community:** ចូលរួមនិងសហគមន៍មិត្តភក្តិរបស់យើងនៅ [Chainlit Discord](https://discord.gg/k73SQ3FyUh) ដើម្បីសួរសំណួរ ចែករំលែកគម្រោងរបស់អ្នក និងភ្ជាប់ទំនក្សងជាមួយអ្នកអភិវឌ្ឍន៍ផ្សេងទៀត! 💬

យើងមិនអាចរង់ចាំឃើញអ្វីដែលអ្នកនឹងបង្កើតជាមួយ Chainlit ទេ! សូមរីករាយក្នុងការសរសេរកូដ! 💻😊

## ផ្ទាំងស្វាគមន៍

ដើម្បីកែប្រែផ្ទាំងស្វាគមន៍ សូមកែសម្រួលឯកសារ `chainlit.md` នៅក្នុងថតដើមនៃគម្រោងរបស់អ្នក។ ប្រសិនបើអ្នកមិនចង់មានផ្ទាំងស្វាគមន៍ វាយឲ្យឯកសារនេះទទេ។
