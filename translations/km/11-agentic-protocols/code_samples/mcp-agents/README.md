# Building Agent-to-Agent Communication Systems with MCP

> TL;DR - бһҸбһҫвҖӢбһўбҹ’бһ“бһҖвҖӢбһўбһ¶бһ…вҖӢбһҹбһ¶бһ„бһҹбһ„бҹӢвҖӢбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ Agent2Agent бһӣбһҫ MCP бһ”бһ¶бһ“бһ‘бҹҒ? бһ”бһ¶бһ“!

MCP бһ”бһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„бһӣбһҫбһҹбһ–бһёбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҠбһҫбһҳбһҡбһ”бһҹбҹӢбһңбһ¶ "бһ•бҹ’бһҸбһӣбҹӢбһ”бһҡбһ·бһ”бһ‘бһ‘бҹ… LLMs"бҹ” бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҗбҹ’бһҳбһёбҹ—бһ‘бһ¶бҹҶбһ„ [бһ…бһҡбһ“бҹ’бһҸбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бһ“бҹ’бһҸбһ”бһ¶бһ“](https://modelcontextprotocol.io/docs/concepts/transports#resumability-and-redelivery), [elicitation](https://modelcontextprotocol.io/specification/2025-06-18/client/elicitation), [sampling](https://modelcontextprotocol.io/specification/2025-06-18/client/sampling), бһ“бһ·бһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ ([progress](https://modelcontextprotocol.io/specification/2025-06-18/basic/utilities/progress) бһ“бһ·бһ„ [resources](https://modelcontextprotocol.io/specification/2025-06-18/schema#resourceupdatednotification)), MCP бһҘбһЎбһјбһңбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҡбһ№бһ„бһҳбһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ„бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһўбҹҒбһ бҹ’бһҹбһ·бһ“вҖӢбһ‘бҹ…вҖӢбһўбҹҒбһ бҹ’бһҹбһ·бһ“бҹ”

## The Agent/Tool Misconception

бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҘбһҡбһ·бһҷбһ¶бһ”бһҗбһўбҹҒбһ бҹ’бһҹбһ·бһ“ (бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһҷбһјбһҡ, бһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ…бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ, бһӣбҹ”), бһҖбһ¶бһҡбһҷбһӣбҹӢбһ…бҹ’бһҡбһЎбҹҶбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһӮбһәбһҗбһ¶ MCP бһҳбһ·бһ“бһҹбһҳбһҹбҹ’бһҡбһ” бһҠбҹ„бһҷбһҹбһ¶бһҡ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҠбҹҶбһ”бһјбһ„бҹ— бһ“бҹғбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһңбһ¶бһӮбһәбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһӮбҹҶбһҡбһјбһҹбҹҶбһҺбһҫ-бһ”бһүбҹ’бһ…бһ”бҹӢбһҹбһ¶бһҳбһүбҹ’бһүбҹ”

бһҖбһ¶бһҡбһҷбһӣбҹӢбһғбһҫбһүбһ“бҹҒбҹҮбһ…бһ¶бһҹбҹӢбһ бһҫбһҷбҹ” бһ–бһ·бһҹбҹ’бһ‘бһ·бһ‘бҹ’бһ’бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲ MCP бһ”бһ¶бһ“бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҒбҹӮбһ–бһёбһҡбһ”бһёбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҠбҹӮбһӣбһ”бһ·бһ‘бһ…бһ“бҹ’бһӣбҹ„бҹҮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһўбҹҠбһёбһҡбһ·бһҷбһ¶бһ”бһҗбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҷбһјбһҡбһ–бҹҒбһӣ:

- **Streaming & Partial Results**: бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһ–бһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·
- **Resumability**: бһўбһҸбһ·бһҗбһ·бһҮбһ“бһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ”бҹӮбһҖбһ…бҹҒбһү
- **Durability**: бһӣбһ‘бҹ’бһ’бһ•бһӣбһ„бһ¶бһҡбһҡбһҹбҹӢбһ“бҹ…бһҖбҹ’бһҡбҹ„бһҷбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒбһЎбһҫбһ„бһңбһ·бһү (бһ§. бһҸбһ¶бһҳбһҡбһҷбҹҲ resource links)
- **Multi-turn**: бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҖбһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· бһҸбһ¶бһҳбһҡбһҷбҹҲ elicitation бһ“бһ·бһ„ sampling

бһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһ…бҹ’бһҡбһҫбһ“ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ MCP бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбҹ”

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҷбҹ„бһ„ бһҷбһҫбһ„бһ“бһ№бһ„бһҷбҹ„бһ„бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ ("tool") бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһӣбһҫбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCPбҹ” бһ“бҹҒбҹҮбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһҳбһ¶бһ“бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһҪбһҷбһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸ MCP client бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҒбһҹбҹ’бһҷбһ»бһ„бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ бһҫбһҷбһўбһ¶бһ…бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ„бҹҮбһ”бһ¶бһ“бҹ”

## What Makes an MCP Tool "Agentic"?

бһҳбһ»бһ“бһ“бһ№бһ„бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸ бһўбҹ„бһҷбһҷбһҫбһ„бһҖбҹҶбһҺбһҸбҹӢ capability бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡвҖӢбһўбҹҒбһ бҹ’бһҹбһ·бһ“вҖӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһјбһҡбҹ”

> бһҷбһҫбһ„бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҗбһ¶бһҮбһ¶бһҮбһ¶бһҜбһҖбһҸбҹ’бһҸбһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҜбһҖбһҡбһ¶бһҮбҹ’бһҷбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бһ”бһ¶бһ“ бһҮбҹҶбһ“бһ¶бһүбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҠбҹӮбһӣбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһ”бһ“бҹ’бһҸбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ…бҹ’бһҡбһҫбһ“бһҠбһ„ бһ¬бһҖбҹӮбһ”бҹ’бһҡбҹӮбһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһ–бҹҒбһӣбһ–бһ·бһҸбҹ”

### 1. Streaming & Partial Results

бһӮбҹҶбһҡбһјбһҹбҹҶбһҺбһҫ-бһ…бһҳбҹ’бһӣбһҫбһҷбһ”бҹ’бһҡбһ–бҹғбһҺбһёбһҳбһ·бһ“бһҠбҹҶбһҺбһҫбһҖбһ¶бһҡ бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһјбһҡбҹ” бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢ:

- бһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһ–бһ·бһҸ
- бһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹҶбһҺбҹӮбһҖбһҖбһҺбҹ’бһҸбһ¶бһӣ

**MCP Support**: бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸ resource бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ…бҹӮбһҖбһ…бһ¶бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹҶбһҺбҹӮбһҖ бһҖбҹ’бһҡбҹ…бһ–бһёбһ“бҹҒбҹҮбһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһҷбһҖбһ…бһ·бһҸбҹ’бһҠбһҷбҹүбһ¶бһ„бһ”бҹ’бһҡбһ»бһ„бһ”бҹ’бһҡбһҷбҹҗбһҸбҹ’бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһј JSON-RPC 1:1 бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҫ/бһ…бһҳбҹ’бһӣбһҫбһҷбҹ”

### 4. Multi-Turn Interactions

бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһҺбҹ’бһҸбһ¶бһӣбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·:

- бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһ¬ бһўбһ“бһ»бһҳбҹҗбһҸбһ–бһёбһҳбһ“бһ»бһҹбҹ’бһҹ
- бһҮбҹҶбһ“бһҪбһҷ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ‘бһ“бҹӢ

**MCP Support**: бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹҶбһ–бҹҒбһүбһҸбһ¶бһҳ sampling (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ–бһё AI) бһ“бһ·бһ„ elicitation (бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһӣбһ–бһёбһҳбһ“бһ»бһҹбҹ’бһҹ)бҹ”

#### Streaming & Progress Updates - Real-time Task Status

Streaming бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһӣбһ–бһ·бһҸбһҖбҹ’бһ“бһ»бһ„бһўбҹҶбһЎбһ»бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбһјбһҡ бһҡбһҖбҹ’бһҹбһ¶бһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҠбһ№бһ„бһ–бһёбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹҶбһҺбҹӮбһҖбҹ”

**Server Implementation (agent sends progress notifications):**

```python
# бһ–бһё server/server.py - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбҹҶбһ–бһ»бһ„бһ•бҹ’бһүбһҫбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–
for i, step in enumerate(steps):
    await ctx.session.send_progress_notification(
        progress_token=ctx.request_id,
        progress=i * 25,
        total=100,
        message=step,
        related_request_id=str(ctx.request_id)
    )
    await anyio.sleep(2)  # бһ…бһҳбҹ’бһӣбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ

# бһҮбһҳбҹ’бһҡбһҫбһҹбһ•бҹ’бһҹбҹҒбһ„: бһҹбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһҠбҹҒбһҸбһӣбһҳбҹ’бһўбһ·бһҸбһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣ
await ctx.session.send_log_message(
    level="info",
    data=f"Processing step {current_step}/{steps} ({progress_percent}%)",
    logger="long_running_agent",
    related_request_id=ctx.request_id,
)
```

**Client Implementation (host receives progress updates):**

```python
# бһ–бһё client/client.py - бһўбһҸбһ·бһҗбһ·бһҮбһ“бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ–бҹҒбһӣбһ–бһ·бһҸ
async def message_handler(message) -> None:
    if isinstance(message, types.ServerNotification):
        if isinstance(message.root, types.LoggingMessageNotification):
            console.print(f"рҹ“Ў [dim]{message.root.params.data}[/dim]")
```

**Event Store Implementation (server holds session state):**

```python
# бһ–бһё server/event_store.py - бһғбҹ’бһӣбһ¶бҹҶбһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҹбһ¶бһҳбһүбҹ’бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бһ„бһ…бһ¶бҹҶ
class SimpleEventStore(EventStore):
    def __init__(self):
        self._events: list[tuple[StreamId, EventId, JSONRPCMessage]] = []
        self._event_id_counter = 0

    async def store_event(self, stream_id: StreamId, message: JSONRPCMessage) -> EventId:
        """Store an event and return its ID."""
        self._event_id_counter += 1
        event_id = str(self._event_id_counter)
        self._events.append((stream_id, event_id, message))
        return event_id

    async def replay_events_after(self, last_event_id: EventId, send_callback: EventCallback) -> StreamId | None:
        """Replay events after the specified ID for resumption."""
        # бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһӮбһ¶бһӣбҹӢ бһ бһҫбһҷбһ…бһ¶бһҖбҹӢбһЎбһҫбһ„бһңбһ·бһү
        for _, event_id, message in self._events[start_index:]:
            await send_callback(EventMessage(message, event_id))

# бһ–бһё server/server.py - бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһғбҹ’бһӣбһ¶бҹҶбһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ‘бҹ…бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹҗбһҷ
def create_server_app(event_store: Optional[EventStore] = None) -> Starlette:
    server = ResumableServer()

    # бһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһҳбҹҗбһҷбһҮбһ¶бһҳбһҪбһҷбһғбҹ’бһӣбһ¶бҹҶбһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ“бҹ’бһҸ
    session_manager = StreamableHTTPSessionManager(
        app=server,
        event_store=event_store,  # бһғбҹ’бһӣбһ¶бҹҶбһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһўбһ¶бһ…бһ”бһ“бҹ’бһҸбһҹбһҳбҹҗбһҷбһ”бһ¶бһ“
        json_response=False,
        security_settings=security_settings,
    )

    return Starlette(routes=[Mount("/mcp", app=session_manager.handle_request)])

# бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ: бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһғбҹ’бһӣбһ¶бҹҶбһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ
event_store = SimpleEventStore()
app = create_server_app(event_store)
```

- **Intelligent Task Decomposition**: Host бһңбһ·бһ—бһ¶бһӮбһҹбҹҶбһҺбһҫбһҹбһ»бҹҶбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһ”бҹҶбһ”бҹӮбһҖбһңбһ¶бһҮбһ¶бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҸбһјбһ…бҹ—бһ‘бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһ–бһ·бһҹбҹҒбһҹбһ•бҹ’бһҹбҹҒбһ„бҹ—
- **Multi-Server Coordination**: Host бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ MCP бһ…бҹ’бһҡбһҫбһ“бҹ— бһҠбҹӮбһӣбһҡбҹҖбһ”бһ…бҹҶбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһ•бҹ’бһҹбҹҒбһ„бҹ—
- **Task State Management**: Host бһҸбһ¶бһҳбһҠбһ¶бһ“бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһңбһ¶бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ dependencies бһ“бһ·бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢ
- **Resilience & Retries**: Host бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹ бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһјбһ бҹ’бһӮбһ·бһ…вҖӢбһ–бҹ’бһҷбһ¶бһҷбһ¶бһҳбһЎбһҫбһ„бһңбһ·бһү бһ бһҫбһҷбһ”бһүбҹ’бһҮбһјбһ“бһҸбҹҶбһҺбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһңбһ·бһүбһ–бҹҒбһӣбһўбҹҒбһ бҹ’бһҹбһ·бһ“вҖӢбһҳбһ·бһ“бһўбһ¶бһ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ
- **Result Synthesis**: Host бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһ…бҹ’бһҡбһҫбһ“бһ‘бҹ…бһҮбһ¶вҖӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷ

Host бһ“бһ№бһ„бһ”бһҸбҹӢбһ”бҹӮбһ“бһ–бһё client бһ’бһҳбҹ’бһҳбһҸбһ¶бһ‘бҹ…бһҮбһ¶бһўбҹ’бһ“бһҖбһҠбһ№бһҖбһ“бһ¶бҹҶбһҳбһ¶бһ“бһ”бһүбҹ’бһүбһ¶ бһ”бһҳбҹ’бһҡбһ»бһ„бһ‘бһ»бһҖбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһ”бҹӮбһҖбһ…бҹӮбһҖ бһ–бҹ„бһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ“бҹ…бһӣбһҫбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ MCP бһҠбһҠбҹӮбһӣбҹ”
