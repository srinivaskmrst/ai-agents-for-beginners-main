# бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҹбҹҠбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI

AI agent frameworks бһӮбһәбһҮбһ¶вҖӢбһңбҹҒбһ‘бһ·бһҖбһ¶бһҹаёӯаёҹаё•а№ҢбһңбҹҗбһҡвҖӢбһҠбҹӮбһӣбһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһү бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI trб»ҹ nГӘnбһҹбһ¶бһҳбһүбҹ’бһүбҹ” бһҹбҹҠбһ»бһҳбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚвҖӢбһ“бһјбһңбһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһҸбһ¶бҹҶбһ„бһҹбҹ’бһҡбһ¶бһҷбһҡбһҪбһ… бһўбһ”бҹ’бһ”бһ”бһҡбһҳбһ¶ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҮбһҪбһҷбһҡбҹҶбһӣбһ¶бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбҹӮбһӣбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

бһҹбҹҠбһ»бһҳбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһўбһ¶бһ…бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһңбһ·бһ’бһёбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AIбҹ” бһ–бһҪбһҖбһңбһ¶бһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһ№бһ„бһұбһҖбһ¶бһҹбһ–бһ„бҹ’бһҡбһёбһҖ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AIбҹ”

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ 

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

- бһҸбһҫбһўбҹ’бһңбһёбһ‘бҹ…бһҮбһ¶бһҹбҹҠбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһ“бһ·бһ„бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһ”бһ¶бһ“бһҒбҹ’бһӣбҹҮ?
- бһҸбһҫбһҖбҹ’бһҡбһ»бһҳбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶вҖӢбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһӣбһҝбһ“ бһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…?
- бһҸбһҫбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…вҖӢбһҡбһңбһ¶бһ„бһҹбҹҠбһ»бһҳбһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣ Microsoft бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ (<a href="https://aka.ms/ai-agents-beginners/ai-agent-service" target="_blank">Azure AI Agent Service</a> бһ“бһ·бһ„ <a href="https://learn.microsoft.com/azure/ai-services/openai/how-to/responses" target="_blank">Microsoft Agent Framework</a>)?
- бһҸбһҫбһҒбҹ’бһүбһ»бҹҶвҖӢбһўбһ¶бһ…бһ•бҹ’бһҹбҹҶбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Azure бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бҹ—бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһ‘бҹҒ, бһ¬бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖ?
- бһҸбһҫ Azure AI Agents service бһӮбһәбһҮбһ¶бһўбҹ’бһңбһё бһ“бһ·бһ„бһңбһ¶бһҮбһҪбһҷбһҒбҹ’бһүбһ»бҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҺбһ¶?

## бһӮбҹ„бһӣбһҠбҹ…бһҖбһ¶бһҡбһҡбҹҖбһ“

бһӮбҹ„бһӣбһҠбҹ…бһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбһәбһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбҹ–

- бһҸбһҪбһ“бһ¶бһ‘бһёбһҡбһ”бһҹбҹӢ AI Agent Frameworks бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ AIбҹ”
- бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫ AI Agent Frameworks бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһҳбһ¶бһ“бһңбһ·бһүбһҶбҹ’бһӣбһ¶бһҸбһңбҹғбҹ”
- бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҠбҹӮбһӣбһҹбҹҠбһ»бһҳбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бҹ”
- бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ Microsoft Agent Framework бһ“бһ·бһ„ Azure AI Agent Serviceбҹ”

## бһҸбһҫбһўбҹ’бһңбһёбһ‘бҹ…бһҮбһ¶бһҹбҹҠбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһ“бһ·бһ„бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһ”бһ¶бһ“бһҒбҹ’бһӣбҹҮ?

бһҹбҹҠбһ»бһҳ AI бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһ»бһҡбһ¶бһҺбһўбһ¶бһ…бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһ”бһүбҹ’бһ…бһјбһӣ AI бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

- **бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“**: AI бһўбһ¶бһ…бһңбһ·бһ—бһ¶бһӮбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸ бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ бһ•бҹ’бһ‘бһ»бһҖбһҳбһ¶бһҸбһ·бһҖбһ¶ бһ“бһ·бһ„бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”
бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ: бһҹбҹҒбһңаёІаёӘаё•бҹ’бһҡбһёбһҳбһҳаёҙа№ҲаёҮбҹӢбһҠбһјбһ…бһҮбһ¶ Netflix бһ”бҹ’бһҡбһҫ AI бһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһ“бһҫбһІбҹ’бһҷбһ—бһ¶бһ–бһҷбһ“бҹ’бһҸ бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҳбһҫбһӣ бһҡбһҪбһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳ бһ“бһ·бһ„бһ–бҹҒбһүбһ…бһ·бһҸбҹ’бһҸбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“бҹ”
- **бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖаӨ°аӨЈ бһ“бһ·бһ„бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–**: AI бһўбһ¶бһ…бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ”бҹҶбһ–бҹҒбһүбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡбһҠбһҠбҹӮбһӣбҹ— бһҡбҹҖбһ”бһ…бҹҶбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ“бһ¶бһ“бһ¶бҹ”

### бһ“бҹҒбҹҮбһӮбһәбһӣбҹ’бһўбһ‘бһҫбһ”бһҺбһ¶бһҹбҹӢ бһҸбҹӮбһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҹбҹҠбһ»бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI?

AI Agent frameworks бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҠбҹӮбһӣбһӣбһҫбһҹбһ–бһёбһҹбҹҠбһ»бһҳ AI бһ‘бһјбһ‘бҹ…бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһ“бһ·бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ‘бһ‘бһҪбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһңбҹӮбһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҠбҹӮбһӣбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбҹ” бһҳбһҖбһҳбһҫбһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбҹҶбһҒбһ¶бһ“бҹ—бһҠбҹӮбһӣбһҹбҹҠбһ»бһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһҪбһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бҹ–

- **бһ”бҹ’бһҡбһҫбһ’бһ¶бһҸбһ»бһҳбҹүбһјбһҢбһ»бһӣ**: SDKs AI бһ•бҹ’бһҸбһӣбҹӢбһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бһҠбһјбһ…бһҮбһ¶ бһҖбһ»бһ„бһҺбҹӮбһӮбһ‘бҹҗбһҡ AI бһ“бһ·бһ„бһҳбҹҒбһҳбҹүбһјбһҡбһё, бһҖбһ¶бһҡ бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ—бһ¶бһҹбһ¶бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· бһ¬ plugin бһҖбһјбһҠ, бһӮбҹҶбһҡбһј prompt бһ“бһ·бһ„бһҠбһ‘бҹғбһ‘бҹҖбһҸбҹ”
- **бһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһ бһҖбһ¶бһҡбһҺбҹҚ**: бһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹ„бһҷбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһё бһ“бһ·бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһ–бһҪбһҖбһңбһ¶бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһҺбҹҚбҹ”
- **бһҡбҹҖбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһ–бһ·бһҸ**: бһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбҹ’бһ”бһ„бҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбҹҖбһ“бһ–бһёбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳ бһ бһҫбһҷбһҖбҹӮбһ”бҹ’бһҡбҹӮбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ–бҹҒбһӣбҹ”

### бһ”бҹ’бһҡбһҫбһ’бһ¶бһҸбһ»бһҳбҹүбһјбһҢбһ»бһӣ

SDKs бһҠбһјбһ…бһҮбһ¶ Microsoft Agent Framework бһ•бҹ’бһҸбһӣбҹӢбһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бһҠбһјбһ…бһҮбһ¶ бһҖбһ»бһ„бһҺбҹӮбһӮбһ‘бҹҗбһҡ AI, бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

**бһҖбҹ’бһҡбһ»бһҳбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶вҖӢбһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…**: бһҖбҹ’бһҡбһ»бһҳбһўбһ¶бһ…вҖӢбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ’бһ¶бһҸбһ»бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹҶбһҡбһјбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“ бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҹбһјбһ“бҹ’бһҷ бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӣбһҝбһ“ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҠбһјбһҡбҹ”

**бһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…бһҖбҹ’бһ“бһ»бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸ**: бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ parser бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһҪбһ…бһҠбһҫбһҳбҹ’бһ”бһёбһҠбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһёбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ, бһҳбҹүбһјбһҢбһ»бһӣбһҳбҹҒбһҳбҹүбһјбһҡбһёбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһ“бһ·бһ„бһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ prompt бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бҹ’бһҸбһҡбһ¶бһӮбһҳбһ“бҹҚбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһ„бҹӢбһ’бһ¶бһҸбһ»бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ–бһёбһҹбһјбһ“бҹ’бһҷбҹ”

**бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠ**. бһҳбһҖбһҳбһҫбһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҳбһҫбһӣбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ Microsoft Agent Framework бһҮбһ¶бһҳбһҪбһҷ `AzureAIProjectAgentProvider` бһҠбһҫбһҳбҹ’бһ”бһёбһІбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ:

``` python
# бһўбһҸбһ·бһҗбһ·бһҮбһ“ Microsoft Agent Framework бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ Python

import asyncio
import os
from typing import Annotated

from agent_framework.azure import AzureAIProjectAgentProvider
from azure.identity import AzureCliCredential


# бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһӮбҹҶбһҡбһјбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡ
def book_flight(date: str, location: str) -> str:
    """Book travel given location and date."""
    return f"Travel was booked to {location} on {date}"


async def main():
    provider = AzureAIProjectAgentProvider(credential=AzureCliCredential())
    agent = await provider.create_agent(
        name="travel_agent",
        instructions="Help the user book travel. Use the book_flight tool when ready.",
        tools=[book_flight],
    )

    response = await agent.run("I'd like to go to New York on January 1, 2025")
    print(response)
    # бһӣбһ‘бҹ’бһ’бһ•бһӣбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ– бһҮбһҫбһ„бһ бҹ„бҹҮбһ бһҫбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹ…бһүбһјбһңбһҷбҹүбһҖбһ“бҹ…бһҗбҹ’бһ„бҹғбһ‘бһё бҹЎ бһҒбҹӮбһҳбһҖбһҡбһ¶ бһҶбҹ’бһ“бһ¶бҹҶ бҹўбҹ бҹўбҹҘ бһ”бһ¶бһ“бһҖбһҖбҹӢбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһҹбһјбһҳбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–! вңҲпёҸрҹ—Ҫ


if __name__ == "__main__":
    asyncio.run(main())
```

## бһҸбһҫбһҳбһ¶бһ“бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ…вҖӢбһҡбһңбһ¶бһ„ Microsoft Agent Framework бһ“бһ·бһ„ Azure AI Agent Service?

бһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһңбһ¶ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҷбһҫбһ„бһҳбһҖбһҹбһҳбҹ’бһӣбһ№бһ„бһ‘бҹ…бһӣбһҫбһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶ бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӮбҹ„бһӣбһҠбҹ…бҹ–

## Microsoft Agent Framework (MAF)

Microsoft Agent Framework бһ•бҹ’бһҸбһӣбҹӢ SDK бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `AzureAIProjectAgentProvider`бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ Azure OpenAI бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбһ¶бҹҶбһ„бһ–бһёбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ Azureбҹ”

**бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ**: бһҹбһ¶бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһҸбҹ’бһҡбҹҖбһҳбһ”бҹ’бһҡбһҫбһҖбһ¶бһҡбһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһ“бҹ…бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҡбһ”бһҹбҹӢ Microsoft Agent Framework:

- **Agents**. бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҳ `AzureAIProjectAgentProvider` бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҲбҹ’бһҳбҹ„бҹҮ бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҺбҹӮбһ“бһ¶бҹҶ бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…:
  - **бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбһ¶бһҡвҖӢбһ”бҹ’бһҡбһҫ** бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ Azure OpenAIбҹ”
  - **бһ бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚ** бһўбһјбһҸбһјбһҳбҹүбһ¶бһ‘бһёбһҖбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҡбһ·бһ”бһ‘бһ“бҹғбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бҹ”
  - **бһҡбһҖбҹ’бһҹбһ¶бһӮ бғЎбғўбғҗбғўбҹҒбһ“бҹғбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶** бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбҹ”

  бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҖбһјбһҠбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ–

    ```python
    import os
    from agent_framework.azure import AzureAIProjectAgentProvider
    from azure.identity import AzureCliCredential

    provider = AzureAIProjectAgentProvider(credential=AzureCliCredential())
    agent = await provider.create_agent(
        name="my_agent",
        instructions="You are a helpful assistant.",
    )

    response = await agent.run("Hello, World!")
    print(response)
    ```

- **Tools**. бһҹбҹҠбһ»бһҳбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҮбһ¶ function бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Python бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһ бҹ…бһ”бһ¶бһ“бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ:

    ```python
    def get_weather(location: str) -> str:
        """Get the current weather for a location."""
        return f"The weather in {location} is sunny, 72\u00b0F."

    agent = await provider.create_agent(
        name="weather_agent",
        instructions="Help users check the weather.",
        tools=[get_weather],
    )
    ```

- **бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ–бһ бһ»бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ**. бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһ„бһ¶бһҡбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһңбһ¶:

    ```python
    planner = await provider.create_agent(
        name="planner",
        instructions="Break down complex tasks into steps.",
    )

    executor = await provider.create_agent(
        name="executor",
        instructions="Execute the planned steps using available tools.",
        tools=[execute_tool],
    )

    plan = await planner.run("Plan a trip to Paris")
    result = await executor.run(f"Execute this plan: {plan}")
    ```

- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ“бҹғбһўбһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҺ Azure**. бһҹбҹҠбһ»бһҳбһ”бҹ’бһҡбһҫ `AzureCliCredential` (бһ¬ `DefaultAzureCredential`) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бҹҶбһ–бҹҒбһү key, бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ API keys бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ”

## Azure AI Agent Service

Azure AI Agent Service бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҗбҹ’бһҳбһё бһҠбҹӮбһӣбһ”бһ¶бһ“бһҺбҹӮбһ“бһ¶бҹҶбһ“бҹ… Microsoft Ignite 2024бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ”бһҸбҹӢбһ”бҹӮбһ“бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ бһҖбһ¶бһҡбһ бҹ…бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ‘бҹ…бһӣбһҫ LLMs бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бһ–вҖӢбһ…бҹҶбһ бһҠбҹӮбһӣбһҮбһ¶ Llama 3, Mistral бһ“бһ·бһ„ Cohereбҹ”

Azure AI Agent Service бһ•бҹ’бһҸбһӣбҹӢбһҷбһ“бҹ’бһҸбһҖбһ¶бһҡбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹӮбһӣбһўбһ¶бһ…бһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

бһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ out-of-the-box бһҮбһ¶бһҳбһҪбһҷ Microsoft Agent Framework бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһҖбҹҶбһ–бһ»бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Public Preview бһ бһҫбһҷбһӮбһ¶бҹҶбһ‘бҹ’бһҡ Python бһ“бһ·бһ„ C# бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Azure AI Agent Service Python SDK бһҷбһҫбһ„бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ:

```python
import asyncio
from azure.identity import DefaultAzureCredential
from azure.ai.projects import AIProjectClient

# бһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһ§бһ”бһҖбһҡбһҺбҹҚ
def get_specials() -> str:
    """Provides a list of specials from the menu."""
    return """
    Special Soup: Clam Chowder
    Special Salad: Cobb Salad
    Special Drink: Chai Tea
    """

def get_item_price(menu_item: str) -> str:
    """Provides the price of the requested menu item."""
    return "$9.99"


async def main() -> None:
    credential = DefaultAzureCredential()
    project_client = AIProjectClient.from_connection_string(
        credential=credential,
        conn_str="your-connection-string",
    )

    agent = project_client.agents.create_agent(
        model="gpt-4o-mini",
        name="Host",
        instructions="Answer questions about the menu.",
        tools=[get_specials, get_item_price],
    )

    thread = project_client.agents.create_thread()

    user_inputs = [
        "Hello",
        "What is the special soup?",
        "How much does that cost?",
        "Thank you",
    ]

    for user_input in user_inputs:
        print(f"# User: '{user_input}'")
        message = project_client.agents.create_message(
            thread_id=thread.id,
            role="user",
            content=user_input,
        )
        run = project_client.agents.create_and_process_run(
            thread_id=thread.id, agent_id=agent.id
        )
        messages = project_client.agents.list_messages(thread_id=thread.id)
        print(f"# Agent: {messages.data[0].content[0].text.value}")


if __name__ == "__main__":
    asyncio.run(main())
```

> A: бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷ Microsoft Agent Framework бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ¶бһ„бһҹбһ„бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһҡбһҪбһ…бһ”бҹ’бһҡбһҫ Azure AI Agent Service бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҠбһ¶бһҖбҹӢбһ…бҹҒбһү бһ“бһ·бһ„бһ–бһ„бҹ’бһҡбһёбһҖбһ–бһҪбһҖбһңбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ” бһңбһ·бһ’бһёбһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбһӣбһҫбһҷбҹ„бһ”бһӣбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбҹ’бһ“бһҖ бһҒбһҺбҹҲбһ–бҹҒбһӣбһҳбһ¶бһ“бһ•бҹ’бһӣбһјбһңбһ…бҹ’бһ”бһ¶бһҹбҹӢбһ‘бҹ…бһҡбһҖбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҠбһӣбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ бһӮбҹ’бһҡбһ¶бһҹбҹ”

бһўбҹ„бһҷбһҷбһҫбһ„бһҹбһ„бҹ’бһҒбҹҒбһ”бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһҖбҹ’бһ“бһ»бһ„бһҸбһ¶бһҡбһ¶бһ„бҹ–

## бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһўбһ¶бһ…бһ•бҹ’бһҹбҹҶбһ§бһ”бһҖбһҡбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Azure бһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһ‘бҹҒ, бһ¬бһҸбһҫбһҒбҹ’бһүбһ»бҹҶбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһ¶бһ…бҹӢбһҠбҹ„бһҷбһЎбҹӮбһҖ?
бһ…бһҳбҹ’бһӣбһҫбһҷвҖӢбһӮбһәбһ”бһ¶бһ‘, бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„вҖӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһҡбһ·бһҹбҹ’бһҗбһ¶бһ“ Azure бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҮбһ¶бһҳбһҪбһҷ Azure AI Agent Service бһ–бһ·бһҹбҹҒбһҹ бһ–бҹ’бһҡбҹ„бҹҮбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһҮбһ¶бһҳбһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ Bing, Azure AI Search, бһ“бһ·бһ„ Azure Functionsбҹ” бһҖбҹҸбһҳбһ¶бһ“бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбҹ’бһҡбһ¶бһӣбһҮбҹ’бһҡбҹ…бһҮбһ¶бһҳбһҪбһҷ Microsoft Foundry бһ•бһ„бһҠбҹӮбһҡбҹ”

The Microsoft Agent Framework бһҖбҹҸбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһҸбһ¶бһҳбһҡбһҷбҹҲ `AzureAIProjectAgentProvider` бһ“бһ·бһ„ Azure identity бһҠбҹӮбһҡ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ бҹ…бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure бһҠбҹ„бһҷвҖӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ–бһёбһ§бһ”бһҖбһҡбһҺбҹҚбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
