## рҹҢұ бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһҳбһ’бҹ’бһҷбһҳбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҳбҹҒбһҡбҹҖбһ“бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ…бҹҶбһҺбһ»бһ…бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ“бҹғбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҸбҹҶбһҺбһ¶бһ„ AIбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҺбһ¶бһҖбҹҸбһ”бһ¶бһ“бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ!

бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ’бҹ’бһҷбһҳбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮбҹ” бһҹбһјбһҳбһ‘бҹ…бһҖбһ¶бһ“бҹӢ [бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](#-multi-language-support)бҹ”

бһ”бһҫбһ“бҹҒбҹҮбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҹбһ¶бһ„бһҹбһ„бҹӢбһҮбһ¶бһҳбһҪбһҷбһӮбҹҶбһҡбһј Generative AI бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҳбһ’бҹ’бһҷбһҳбһҹбһ·бһҖбҹ’бһҹбһ¶ [Generative AI бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ](https://aka.ms/genai-beginners) бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбҹҒбһҡбҹҖбһ“ бҹўбҹЎбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһ¶бһ„бһҹбһ„бҹӢбһҮбһ¶бһҳбһҪбһҷ GenAIбҹ”

бһҖбһ»бҹҶбһ—бҹ’бһӣбҹҒбһ… [бһ•бҹ’бһҖбһ¶бһҷ (рҹҢҹ) бһ‘бһ»бһҖбһІбҹ’бһҷбһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) бһ“бһ·бһ„ [fork бһғбҹ’бһӣбһ¶бҹҶбһ„бһ“бҹҒбҹҮ](https://github.com/microsoft/ai-agents-for-beginners/fork) бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһјбһҠбҹ”
