# бһҡбһ…бһ“бһ¶бһҸбһҳбҹ’бһҡбҹҖбһ”бһ•бҹӮбһ“бһҖбһ¶бһҡ

## бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ

* бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһӮбҹ„бһӣбһҠбҹ…бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҠбҹӮбһӣбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“бҹ”
* бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹӮбһӣбһҮбһҝбһҮбһ¶бһҖбҹӢбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһўбһ¶бһ…бһўбһ¶бһ“бһҠбҹ„бһҷбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ”
* бһўбһ“бһ»бһңбһҸбҹ’бһҸбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҠбҹӮбһӣбһ”бһҫбһҖбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбҹ„бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҢбһёбһҺбһ¶бһҳбһ·бһ…бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹӮбһӣбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ”

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡвҖӢбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбҹ–

* бһҡбһҖбһғбһҫбһү бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһӮбҹ„бһӣбһҠбҹ…бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһҺбһ¶бһ„ AI бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһңбһ¶бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҗбһ¶бһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбҹ”
* бһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ‘бҹ…бһҮбһ¶бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ„бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“бһ бһҫбһҷбһҡбҹҖбһ”бһ…бҹҶбһ–бһҪбһҖбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”
* бһҡбҹҖбһ”бһ…бҹҶвҖӢбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһҳбһҪбһҷбһ§бһ”бһҖбһҡбһҺбҹҚбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң (бһ§. бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ¬бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ) бһ–бһёбһ…бҹҶбһ–бҹ„бҹҮбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҠбҹӮбһӣбһ–бһҪбһҖбһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ бһ бһҫбһҷбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһҳбһ·бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбҹ”
* бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбһ„бҹ— бһңбһ¶бһҹбҹӢбһ”бһҡбһ·бһҳбһ¶бһҺбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһҳбһ“бҹ’бһҸбҹ’бһҡбһёбһӣбһҫбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бҹ”

## бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһӮбҹ„бһӣбһҠбҹ…бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…

бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһӣбҹ„бһҖбһ–бһ·бһҸбһҮбһ¶бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһ–бҹҒбһҖбһҳбһ·бһ“бһўбһ¶бһ…бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһҸбҹӮбһҳбһҪбһҷбһ”бһ¶бһ“бҹ” бһ бһҫбһҷбһ—бһ¶бһҹбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҶбһҺбһ¶бһ„ AI бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҠбҹҸбһҒбҹ’бһӣбһёбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҺбҹӮбһ“бһ¶бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡ бһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһңбһ¶бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ, бһҹбһјбһҳбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһӮбҹ„бһӣбһҠбҹ…бҹ–

    "бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ–бһёбһҡбһ”бһёбһҗбҹ’бһ„бҹғбҹ”"

бһ”бһҫбһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһңбһ¶бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ бһўбһ¶бһ…бһ“бһ№бһҖбһҹбҹ’бһҡбһҳбҹғбһҗбһ¶бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ” бһӮбҹ„бһӣбһҠбҹ…бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһ„бһ“бҹҒбҹҮ бһҸбҹҶбһҺбһ¶бһ„ (бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбһ”бһ»бһӮбҹ’бһӮбһӣбһҺбһ¶бһҳбһҪбһҷ) бһўбһ¶бһ…бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ”бһ¶бһ“бһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһҠбһјбһ…бһҮбһ¶бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҹбһ–бҹ’бһңбһҗбҹ’бһ„бҹғбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹбһўбһ¶бһҖбһ¶бһҹбһ…бһҡбһҺбҹҚ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһўбһ¶бһ бһ¶бһҡбһҠбҹ’бһӢбһ¶бһ“ бһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бһ“бһ·бһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚбҹ”

### бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…

бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ’бҹҶбһ¬бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ’бҹҶбһЎбһҫбһ„бһ“бҹ…бһ–бҹҒбһӣбһ”бҹҶбһ”бҹӮбһҖбһҮбһ¶бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҡбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”
бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹҶбһ”бҹӮбһҖбһӮбҹ„бһӣбһҠбҹ…бһ‘бҹ…бһҮбһ¶бҹ–

Magentic One - бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹҶбһҺбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһӣбһҖбҹ’бһҒбһҺбҹҲбһ‘бһјбһ‘бҹ…бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ‘бһҪбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһўбһҹбҹ’бһ…бһ¶бһҡбҹ’бһҷбһӣбһҫбһ”бҹҒбһ“бһ…бһҳбһ¶бһӮбҹ’бһҡбһ¶бһҹбҹӢбһҸбҹҶбһҺбһ¶бһ„бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһҜбһҖбһҹбһ¶бһҡбһҷбҹ„бһ„: <a href="https://www.microsoft.com/research/articles/magentic-one-a-generalist-multi-agent-system-for-solving-complex-tasks" target="_blank">Magentic One</a>. бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бһёбһҳбһҪбһҷбҹ— бһ бһҫбһҷбһ”бҹ’бһҡбһӮбһӣбҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ‘бҹ…бһҠбһӣбҹӢбһҸбҹҶбһҺбһ¶бһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбҹ” бһҮбһ¶бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡ бһўбҹ’бһ“бһҖбһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡбһ“бҹҒбҹҮбһҖбҹҸбһ”бҹ’бһҡбһҫбһҷбһ“бҹ’бһҠбһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“бһ“бҹғбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ… бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡбһЎбһҫбһ„бһңбһ·бһүбһ‘бҹ…бһҸбһ¶бһҳбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһјбһңбҹ”
