## бһӮбҹҶбһ“бһ·бһҸбһҹбҹҶбһҒбһ¶бһ“бҹӢ

### Structured Planning with Type-Safe Models

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бҹ’бһҡбһ”бҹ’бһҡбһҫбһҗбҹ’бһ“бһ¶бһҖбҹӢ C# бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ“бҹғбһӣбһ‘бҹ’бһ’бһ•бһӣбһ•бҹӮбһ“бһҖбһ¶бһҡбҹ–

```csharp
public class Plan
{
    [JsonPropertyName("assigned_agent")]
    public string? Assigned_agent { get; set; }

    [JsonPropertyName("task_details")]
    public string? Task_details { get; set; }
}

public class TravelPlan
{
    [JsonPropertyName("main_task")]
    public string? Main_task { get; set; }

    [JsonPropertyName("subtasks")]
    public IList<Plan> Subtasks { get; set; }
}
```

### JSON Schema for Structured Outputs

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһІбҹ’бһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ“бһ№бһ„бһҹбҹ’бһӮбһёбһҳбһ¶ TravelPlan:

```csharp
ChatClientAgentOptions agentOptions = new(name: AGENT_NAME, instructions: AGENT_INSTRUCTIONS)
{
    ChatOptions = new()
    {
        ResponseFormat = ChatResponseFormatJson.ForJsonSchema(
            schema: AIJsonUtilities.CreateJsonSchema(typeof(TravelPlan)),
            schemaName: "TravelPlan",
            schemaDescription: "Travel Plan with main_task and subtasks")
    }
};
```

### Planning Agent Instructions

бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбһјбһ…бһҮбһ¶бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ бһҠбһ¶бһҖбҹӢбһ…бҹ„бһӣбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҡбһ„бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ–

- FlightBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ бҹ„бҹҮбһ бһҫбһҡ
- HotelBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ
- CarRental: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбһҪбһӣбһЎбһ¶бһ“ бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҮбһҪбһӣбһЎбһ¶бһ“
- ActivitiesBooking: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҖбҹӢбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– бһ“бһ·бһ„бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–
- DestinationInfo: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһ‘бһёбһҖбһ“бҹ’бһӣбҹӮбһ„
- DefaultAgent: бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҹбҹҶбһҺбһҫбһ‘бһјбһ‘бҹ…
