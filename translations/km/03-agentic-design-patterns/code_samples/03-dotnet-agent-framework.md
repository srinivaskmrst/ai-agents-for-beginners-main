# рҹҺЁ бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶ Agentic бһҮбһ¶бһҳбһҪбһҷ GitHub Models (.NET)

## рҹ“Ӣ бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҸбһ¶бһҳбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ (enterprise-grade) бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһ„бҹӢбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҶбҹ’бһӣбһ¶бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Microsoft Agent Framework бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ .NET бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ GitHub Modelsбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһӣбҹҶбһ“бһ¶бҹҶбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲ бһ“бһ·бһ„бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ, бһ„бһ¶бһҷбһҹбҹҶбһҡбһ¶бһ”бҹӢбһҗбҹӮбһ‘бһ¶бҹҶ бһ“бһ·бһ„бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бҹ”

### бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

- рҹҸӯ **Factory Pattern**: бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһјбһӣбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ—бһ¶бһ–
- рҹ”§ **Builder Pattern**: бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҠбҹ„бһҷбһҡбһӣбһјбһ“
- рҹ§ө **Thread-Safe Patterns**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҮбһ¶бһҳбһҪбһҷбһ…бҹ’бһҡбһҫбһ“бһ…бҹ’бһҡбһҖбһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–
- рҹ“Ӣ **Repository Pattern**: бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһўбҹ„бһҷбһҳбһ¶бһ“бһҡбһ”бҹҖбһ”

## рҹҺҜ бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ–бһ·бһҹбҹҒбһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ .NET

### бһӣбһҖбҹ’бһҒбһҺбҹҲбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

- **Strong Typing**: бһҖбһ¶бһҡбһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһ–бҹҒбһӣбһҖ compiling бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡ IntelliSense
- **Dependency Injection**: бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҳбһҪбһҷбһ’бһ»бһ„ DI бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
- **Configuration Management**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡвҖӢбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ IConfiguration бһ“бһ·бһ„ Options patterns
- **Async/Await**: бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһҹбҹҠбһёбһ“бһҒбһ¶бһ„бһҳбһ»бһҒ

### бһӣбҹҶбһ“бһ¶бҹҶбһҠбҹӮбһӣбһҡбһҪбһ…бһҡбһ¶бһӣбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбһ¶бһҖбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ

- **Logging Integration**: бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ ILogger бһ“бһ·бһ„бһҖбһ¶бһҡбһ…бһ»бһ…бһ”бҹ’бһӣбһ»бһҖбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷ
- **Health Checks**: бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһҒбһ—бһ¶бһ– бһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҡбһ¶бһ”бҹӢ
- **Configuration Validation**: бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„бһ“бһ·бһҷбһҳбһ“бҹҗбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ
- **Error Handling**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһҡбһҺбһёбһҖбһҳбҹ’бһҡбһ·бһҸбһҖбҹҶбһ бһ»бһҹбһҮбһ¶бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳ

## рҹ”§ бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹ

### бһ’бһ¶бһҸбһ»бһҹбҹ’бһ“бһјбһӣ .NET

- **Microsoft.Extensions.AI**: бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ AI бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷ
- **Microsoft.Agents.AI**: бһҹбҹҠбһ»бһҳбһўбһ¶бһҮбһёбһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһ¶бһҸбҹӢбһ…бҹӮбһ„бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҖбҹ’бһ“бһ»бһ„бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ
- **GitHub Models Integration**: бһӣбҹҶбһ“бһ¶бҹҶбһўбһҸбһ·бһ—бһ¶бһ–бһҖбҹ’бһҡбҹ„бһҳбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ API client бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢ
- **Configuration System**: appsettings.json and environment integration

## рҹҸ—пёҸ бһӣбҹҶбһ“бһ¶бҹҶбһҡбһ…бһ“бһ¶бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһү

### 1. **Creational Patterns**

- **Agent Factory**: бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҖбһҺбҹ’бһҸбһ¶бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҜбһҖбһ—бһ¶бһ–
- **Builder Pattern**: API бһҡбһ¶бһ„бһҡбһӣбһјбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү
- **Singleton Pattern**: бһҖбһ¶бһҡбһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһ’бһ“бһ’бһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ
- **Dependency Injection**: бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ¶бһ” бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ

### 2. **Behavioral Patterns**

- **Strategy Pattern**: бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ¶бһ…бһ…бҹӮбһҖбһ”бҹ’бһҠбһјбһҡбһ”бһ¶бһ“
- **Command Pattern**: бһўбһ“бһ»бһ”бһҠбһ·бһ…бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҠбҹӮбһӣбһ”бһҺбҹ’бһҸбһјбһӣбһ‘бһ¶бҹҶбһ„ undo/redo
- **Observer Pattern**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһёбһ бҹ’бһңбһҹбҹ’бһҸбһ¶бһҷбһҠбҹӮбһӣбһ”бһҫбһҖбһҠбҹ„бһҷбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ
- **Template Method**: бһңбһёбһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ”бһүбҹ’бһҮбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ—бһ¶бһ–бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҠбҹӮбһӣбһ”бһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ

### 3. **Structural Patterns**

- **Adapter Pattern**: бһҹбҹ’бһҡбһ‘бһ¶бһ”бҹӢбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ API GitHub Models
- **Decorator Pattern**: бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһўбҹҒбһ бҹ’бһҹбһ·бһ“
- **Facade Pattern**: бһҳбһ»бһҒбһ„бһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҹбһ¶бһҳбһүбҹ’бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹҒбһ бҹ’бһҹбһ·бһ“
- **Proxy Pattern**: бһҖбһ¶бһҡбһӣбҹ„бһҸбһ…бһјбһӣбһҷбһәбһҸ бһ“бһ·бһ„ caching бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–

## рҹ“ҡ бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҡбһ…бһ“бһ¶бһҡбһ”бһҹбҹӢ .NET

### бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚ SOLID

- **Single Responsibility**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһҝбһ„бһ”бһ“бҹ’бһӣбһ¶бһҹбҹӢбһҳбһ¶бһ“бһ”бҹҒбһҹбһҖбһҖбһҳбҹ’бһҳбһҳбһҪбһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ
- **Open/Closed**: бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһҖбҹӮбһ”бҹ’бһҡбҹӮ
- **Liskov Substitution**: бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһўбҹҠбһёбһ•бҹҒбһҹ
- **Interface Segregation**: бһўбҹҠбһёбһ•бҹҒбһҹбһ•бҹ’бһҸбҹ„бһҸбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡ бһ“бһ·бһ„бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ
- **Dependency Inversion**: бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһўбһ¶бһҺбһ¶бһ…бһҖбҹ’бһҡ бһҳбһ·бһ“бһҳбҹӮбһ“бһӣбһҫбһҡбһ”бһҹбҹӢбһ–бһ·бһҸ

### бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһҹбҹ’бһўбһ¶бһҸ

- **Domain Layer**: бһҹбҹ’бһ“бһјбһӣбһўбҹҒбһ бҹ’бһҹбһ·бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ§бһ”бһҖбһҡбһҺбҹҚ
- **Application Layer**: бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ…бһ¶бһҸбҹӢбһўбҹҒбһ бҹ’бһҹбһ·бһ“ бһ“бһ·бһ„ workflow
- **Infrastructure Layer**: бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ GitHub Models бһ“бһ·бһ„бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…
- **Presentation Layer**: бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ“бһ·бһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ…бһҳбҹ’бһӣбһҫбһҷ

## рҹ”’ бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ бһӮбҹ’бһҡбһ¶бһҹ

### бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

- **Credential Management**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһјбһ“бһҹбҹ„ API бһҠбҹ„бһҷбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷ IConfiguration
- **Input Validation**: бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ·бһ„ annotation
- **Output Sanitization**: бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ…бһҳбҹ’бһӣбһҫбһҷ бһ“бһ·бһ„бһ…бһҳбҹ’бһҡбһ¶бһүбҹӢ
- **Audit Logging**: бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡвҖӢбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷ

### бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–

- **Async Patterns**: бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ I/O бһҳбһ·бһ“бһ”бһүбҹ’бһҲбһ”бҹӢ
- **Connection Pooling**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ HTTP client бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **Caching**: бһҖбһ¶бһҡбһҒбҹҒбһҖбһ…бһҳбҹ’бһӣбһҫбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **Resource Management**: бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ бһ“бһ·бһ„бһҹбһҳбҹ’бһўбһ¶бһҸбһ’бһ“бһ’бһ¶бһ“бһ”бһ¶бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң

### бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ–бһ„бҹ’бһҡбһёбһҖ

- **Thread Safety**: бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһўбҹҒбһ бҹ’бһҹбһ·бһ“бһҮбһ¶бһҳбһҪбһҷбһ…бҹ’бһҡбһҫбһ“бһ’бҹ’бһ“бһ¶бһҖбҹӢ
- **Resource Pooling**: бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ’бһ“бһ’бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **Load Management**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһҸбҹ’бһҡбһ¶ бһ“бһ·бһ„бһҖбһ¶бһҡбһ‘бһ”бҹӢбһҹбҹ’бһҖбһ¶бһҸбҹӢбһ‘бһҳбҹ’бһ„бһ“бҹӢбһҖбҹ’бһҡбҹ„бһҷ
- **Monitoring**: бһңбһ·бһҳбһ¶бһҸбҹ’бһҡ бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһ»бһҒбһ—бһ¶бһ–

## рҹҡҖ бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ

- **Configuration Management**: бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҸбһ¶бһҳбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢ
- **Logging Strategy**: бһҖбһ¶бһҡбһ…бһ»бҹҮбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһҮбһ¶бһҳбһҪбһҷ correlation IDs
- **Error Handling**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹбһҮбһ¶бһҹбһҖбһӣбһҠбҹ„бһҷбһҳбһ¶бһ“бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң
- **Monitoring**: Application insights бһ“бһ·бһ„ counters бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–

```csharp
#!/usr/bin/dotnet run

#:package Microsoft.Extensions.AI@10.*
#:package Microsoft.Agents.AI.OpenAI@1.*-*

using System.ClientModel;
using System.ComponentModel;

using Microsoft.Agents.AI;
using Microsoft.Extensions.AI;

using OpenAI;

// Tool Function: Random Destination Generator
// This static method will be available to the agent as a callable tool
// The [Description] attribute helps the AI understand when to use this function
// This demonstrates how to create custom tools for AI agents
[Description("Provides a random vacation destination.")]
static string GetRandomDestination()
{
    // List of popular vacation destinations around the world
    // The agent will randomly select from these options
    var destinations = new List<string>
    {
        "Paris, France",
        "Tokyo, Japan",
        "New York City, USA",
        "Sydney, Australia",
        "Rome, Italy",
        "Barcelona, Spain",
        "Cape Town, South Africa",
        "Rio de Janeiro, Brazil",
        "Bangkok, Thailand",
        "Vancouver, Canada"
    };

    // Generate random index and return selected destination
    // Uses System.Random for simple random selection
    var random = new Random();
    int index = random.Next(destinations.Count);
    return destinations[index];
}

// Extract configuration from environment variables
// Retrieve the GitHub Models API endpoint, defaults to https://models.github.ai/inference if not specified
// Retrieve the model ID, defaults to openai/gpt-5-mini if not specified
// Retrieve the GitHub token for authentication, throws exception if not specified
var github_endpoint = Environment.GetEnvironmentVariable("GH_ENDPOINT") ?? "https://models.github.ai/inference";
var github_model_id = Environment.GetEnvironmentVariable("GH_MODEL_ID") ?? "openai/gpt-5-mini";
var github_token = Environment.GetEnvironmentVariable("GH_TOKEN") ?? throw new InvalidOperationException("GH_TOKEN is not set.");

// Configure OpenAI Client Options
// Create configuration options to point to GitHub Models endpoint
// This redirects OpenAI client calls to GitHub's model inference service
var openAIOptions = new OpenAIClientOptions()
{
    Endpoint = new Uri(github_endpoint)
};

// Initialize OpenAI Client with GitHub Models Configuration
// Create OpenAI client using GitHub token for authentication
// Configure it to use GitHub Models endpoint instead of OpenAI directly
var openAIClient = new OpenAIClient(new ApiKeyCredential(github_token), openAIOptions);

// Define Agent Identity and Comprehensive Instructions
// Agent name for identification and logging purposes
var AGENT_NAME = "TravelAgent";

// Detailed instructions that define the agent's personality, capabilities, and behavior
// This system prompt shapes how the agent responds and interacts with users
var AGENT_INSTRUCTIONS = """
You are a helpful AI Agent that can help plan vacations for customers.

Important: When users specify a destination, always plan for that location. Only suggest random destinations when the user hasn't specified a preference.

When the conversation begins, introduce yourself with this message:
"Hello! I'm your TravelAgent assistant. I can help plan vacations and suggest interesting destinations for you. Here are some things you can ask me:
1. Plan a day trip to a specific location
2. Suggest a random vacation destination
3. Find destinations with specific features (beaches, mountains, historical sites, etc.)
4. Plan an alternative trip if you don't like my first suggestion

What kind of trip would you like me to help you plan today?"

Always prioritize user preferences. If they mention a specific destination like "Bali" or "Paris," focus your planning on that location rather than suggesting alternatives.
""";

// Create AI Agent with Advanced Travel Planning Capabilities
// Initialize complete agent pipeline: OpenAI client вҶ’ Chat client вҶ’ AI agent
// Configure agent with name, detailed instructions, and available tools
// This demonstrates the .NET agent creation pattern with full configuration
AIAgent agent = openAIClient
    .GetChatClient(github_model_id)
    .CreateAIAgent(
        name: AGENT_NAME,
        instructions: AGENT_INSTRUCTIONS,
        tools: [AIFunctionFactory.Create(GetRandomDestination)]
    );

// Create New Conversation Thread for Context Management
// Initialize a new conversation thread to maintain context across multiple interactions
// Threads enable the agent to remember previous exchanges and maintain conversational state
// This is essential for multi-turn conversations and contextual understanding
AgentThread thread = agent.GetNewThread();

// Execute Agent: First Travel Planning Request
// Run the agent with an initial request that will likely trigger the random destination tool
// The agent will analyze the request, use the GetRandomDestination tool, and create an itinerary
// Using the thread parameter maintains conversation context for subsequent interactions
await foreach (var update in agent.RunStreamingAsync("Plan me a day trip", thread))
{
    await Task.Delay(10);
    Console.Write(update);
}

Console.WriteLine();

// Execute Agent: Follow-up Request with Context Awareness
// Demonstrate contextual conversation by referencing the previous response
// The agent remembers the previous destination suggestion and will provide an alternative
// This showcases the power of conversation threads and contextual understanding in .NET agents
await foreach (var update in agent.RunStreamingAsync("I don't like that destination. Plan me another vacation.", thread))
{
    await Task.Delay(10);
    Console.Write(update);
}
```
