# бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹ„бһ„ (AI Agentic Design Principles)

## бһҳбҹҒбһҡбҹҖбһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“

бһҳбһ¶бһ“бһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбһ·бһҸбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AIбҹ” бһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ—бһ¶бһ–бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӮбһәбһҮбһ¶бһҳбһ»бһҒбһ„бһ¶бһҡбһҳбһҪбһҷ бһҳбһ·бһ“бһҳбҹӮбһ“бһ…бһ»бҹҮбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹҶбһ бһ»бһҹбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бҹҒ бһҖбҹҸбһҠбҹ„бһҷ бһңбһ¶бһҖбҹҸбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһңбһ·бһҹбҹ’бһңбһҖбһҡбһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһҳбһ¶бһ“бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбһ№бһ„бһҗбһ¶бһҸбҹ’бһҡбһјбһңбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҺбһ¶бҹ” бһҷбһҫбһ„бһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһҡбһ»бһҳбһ“бҹғбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸ UX бһҠбҹӮбһӣбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһҳбһ“бһ»бһҹбҹ’бһҹ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһ•бҹ’бһҠбҹ„бһҸбһӣбһҫбһўбһҸбһ·бһҗбһ·бһҮбһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһҒбһ¶бһ„бһ“бҹҒбҹҮбһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҹбҹҶбһҺбһ„бҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбҹүбһ¶бһ„бһҳбҹүбһ¶бһҖбһҺбһ¶бһҳбһҪбһҷбһ‘бҹҒ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

бһҠбҹ„бһҷбһ‘бһјбһ‘бҹ… бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһӮбһҪбһҡбһҸбҹӮ:

- бһ–бһ„бҹ’бһҡбһёбһҖбһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбһ“бһ»бһҹбҹ’бһҹ (бһӮбҹҶбһ“бһ·бһҸбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҖбһ¶бһҡбһҷбһӣбҹӢбһ…бҹҶбһ–бҹ„бҹҮ, бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶, бһўбһјбһҸбһјбһҳбҹүбһ¶бһҹбҹ’бһҷбһ»бһ„ бһ¬бһ•бҹ’бһҹбҹҒбһ„бҹ—)
- бһ”бҹҶбһ–бҹҒбһүбһ…бһ“бҹ’бһӣбҹ„бҹҮбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„ (бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҒбҹ’бһүбһ»бҹҶбһ‘бһ¶бһ“бҹӢбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҠбҹӮбһ“бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„ бһҖбһ¶бһҡбһ”бһҖбһ”бҹ’бһҡбҹӮ бһҠбһјбһ…бһҮбһ¶)
- бһҮбһҪбһҷбһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҷбһҫбһ„бһҮбһ¶бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһ“бһ·бһҷбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһҠбһ‘бҹғ
- бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҷбһҫбһ„бһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„бһҮбһ¶бһҖбҹҶбһҺбҹӮбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ (бһ§. бһӮбҹ’бһҡбһјбһҮбһёбһңбһ·бһҸ/бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһҡбҹҲбһҖбһ¶бһҡ, бһҮбһҪбһҷбһҷбһҫбһ„бһҡбҹҖбһ“бһҮбҹҶбһ“бһ¶бһүбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢ, бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһҡбһ№бһ„бһҳбһ¶бҹҶ бһ бһҫбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—)

## бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢ

- бһҸбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹ„бһ„бһӮбһәбһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ
- бһҸбһҫбһҳбһ¶бһ“бһҳбһ¶бһӮбҹҢбһ¶бһҺбһ¶бһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҒбһҺбҹҲбһ–бҹҒбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһҒбһ¶бһ„бһ“бҹҒбҹҮ
- бһҸбһҫбһҳбһ¶бһ“бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҺбһ¶бһҒбҹ’бһӣбҹҮбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһҒбһ¶бһ„бһ“бҹҒбҹҮ

## бһӮбҹ„бһӣбһҠбҹ…бһҖбһ¶бһҡбһҡбҹҖбһ“

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһўбһ¶бһ…:

1. бһ–бһ“бҹ’бһҷбһӣбҹӢбһҗбһ¶бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹ„бһ„бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ
2. бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһҳбһ¶бһӮбҹҢбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
3. бһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ

## бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҷбҹ„бһ„

### бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ‘бһ·бһҹбһҠбҹ…бһ…бҹҶбһ–бҹ„бҹҮбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҷбһҫбһ„бһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҖбҹ’бһ“бһ»бһ„бһ–бһ·бһ—бһ–бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһҢбһёбһҮбһёбһҗбһӣЦү

- **бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ бһҳбһ·бһ“бһҳбҹӮбһ“бһ”бһ„бҹ’бһҖбһ”бҹӮбһҖбһ‘бҹҒ** вҖ“ бһҮбһҪбһҷбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбҹӮбһӣбһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“ бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһ“бһ·бһ„бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”
- бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһҪбһҷбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„ бһ“бһ·бһ„бһҳбһ“бһ»бһҹбҹ’бһҹбҹ”
- бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһ…бҹҒбһүбһҳбһ“бһ»бһҹбҹ’бһҹбһұбһҖбһ¶бһҹбһ“бҹ…бһҮбһ·бһҸбһӮбҹ’бһ“бһ¶бһҷбһјбһҡбһҳбһ·бһ“бһҳбҹӮбһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһҷбһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҶбһ“бһҪбһҹбһ¬бһ”бһ„бҹ’бһҒбһјбһ…бһҗаёІаё„аёҷбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҒбҹ”
- **бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбҹҚбһўбһ¶бһ…бһӣбһ¶бһ„бһғбһҫбһүбһ”бһ¶бһ“бһҮбһ¶бһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮ** вҖ“ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ…бһҳбҹ’бһ”бһ„бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ бһҫбһҷбһҮбһ¶бһҖбһ¶бһҡбһ–бҹҒбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ‘бһ“бҹ’бһҹбһ¶бһҷбһҸбһ·бһ…бһҸбһҪбһ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһңбһ¶бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ“бһ·бһ„бһҹбһҳбһҡбһҳбҹ’бһҷбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҖбһ¶бһ“бҹӢбһҸбҹӮбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҡбһҖбһғбһҫбһү бһ“бһ·бһ„бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһ¬бһңбҹҒбһ‘бһ·бһҖбһ¶бһҺбһ¶бҹ—бһҖбҹҸбһ”бһ¶бһ“бҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бһүбҹ’бһ…бһјбһӣ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһҡбһ”бҹҖбһ” (бһҹбһҳбҹ’бһӣбҹҒбһ„, бһҖбһ¶бһҡбһ“бһ·бһҷбһ¶бһҷ, бһўбһҸбҹ’бһҗбһ”бһ‘, бһӣбҹ”)
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһ•бҹ’бһ‘бҹҒбһҡбһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹбһҡбһңбһ¶бһ„бһ•бҹ’бһ“бҹӮбһҖбһҳбһ»бһҒ бһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ; бһҡбһңбһ¶бһ„бһҖбһ¶бһҡбһ‘бһ”бҹӢбһҸбһ¶бһҳбһҳбһ»бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”, бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҸбһ¶бһҳбһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ”бҹӮбһ”бһҺбһ¶бһҳбһҪбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бҹ’бһ“бҹӮбһҖбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ бһҳбһ¶бһ“бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһўбһ¶бһ…бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”

### бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһ–бҹҒбһӣбһңбҹҒбһӣбһ¶)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһҖбһ¶бһҡбһ„бһ¶бһҡбһӣбһҫбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹ” бһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһ‘бһ·бһҹбһҠбҹ…бһ…бҹҶбһ–бҹ„бҹҮбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҷбһҫбһ„бһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһҳбҹ’бһҠбһ„бһ”бһ»бһҡбһ¶бһҺ бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“ бһ“бһ·бһ„бһўбһ“бһ¶бһӮбһҸбҹ”

- **бһўбһҸбһёбһҸбһҖбһ¶бһӣ**бҹ– бһҖбһ¶бһҡбһ–бһ·бһ…бһ¶бһҡбһҺбһ¶бһ–бһёбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бһ·бһ„бһ”бһҡбһ·бһ”бһ‘бһҠбҹӮбһӣбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ•бһ„бһҠбҹӮбһҡбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһ–бһёбһҳбһ»бһ“ бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһ”бһ‘бһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ бһҳбһ“бһ»бһҹбҹ’бһҹ бһ¬бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ–бһёбһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҖбһ“бҹ’бһӣбһ„бһҳбһҖ бһ бһҫбһҷбһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһўбһ“бһ»бһҹбҹ’бһҹбһ¶бһңбһҡбһёбһҷбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһ…бһјбһӣбһҡбһҪбһҳбһҮбһ¶бһҳбһҪбһҷбһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ”
- **бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“**бҹ– бһҖбһ¶бһҡбһ”бһ„бҹ’бһңбһ·бһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбһ¶бһ„бһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ¶бһ“бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбһ¶бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷбһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһ–бҹҒбһӣбһҳбһ¶бһ“бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚ, бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ·бһ“бһҸбҹ’бһҡбһ№бһҳбһҸбҹӮбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҹбҹ’бһҗбһ·бһҸбһҹбҹ’бһҗбҹҒбһҡ бһ¬бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ•бҹ’бһӣбһјбһңбһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ‘бҹҒбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбһүбҹ’бһүбһ¶бһ•бҹ’бһ‘бһ»бһҖбһ…бһӣбһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһүбһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһҹбһҳбһҡбһҳбҹ’бһҷбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһүбҹ’бһҮбһјбһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ”бһҡбһ·бһ‘бһ·бһҹбһ“бҹҚ бһ—бһ¶бһ–бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҹбһ„бҹ’бһӮбһҳ бһ“бһ·бһ„бһңбһ”бҹ’бһ”бһ’бһҳбҹҢ бһ“бһ·бһ„бһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһ…бҹҒбһҸбһ“бһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”
  - бһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҮбһ¶вҖӢбһҠбҹҶбһҺбһҫбһҡбһ”бһ“бҹ’бһҸбҹ— бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһҫбһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбһҷбһәбһҸбһҷбҹүбһ¶бһң бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҠбһӣбҹӢбһўбҹҶбһҺбһ¶бһ…бһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бҹ”
- **бһўбһ“бһ¶бһӮбһҸ**бҹ– бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһ§бһ”бһҖбһҡбһҺбҹҚ бһңбҹҒбһ‘бһ·бһҖбһ¶ бһ“бһ·бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһӮбһ„бҹӢбһ—бһ¶бһ–бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһўбһ¶бһ…бһҖбҹӮбһ”бһ¶бһ“бһҠбҹ„бһҷбһҹбҹҒбһҡбһёбҹ”
  - бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ‘бҹҶбһҡбһ„бҹӢ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбҹ”

### бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“)

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”

- **бһ‘бһ‘бһҪбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһ—бһ¶бһ–бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҖбһҹбҹ’бһҗбһ¶бһ”бһ“бһ¶бһұбҹ’бһҷбһҳбһ¶бһ“бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ**бҹ”
  - бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбһҪбһҷбһ“бҹғбһ—бһ¶бһ–бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҖбҹ’бһ“бһ»бһ„бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһӮбһәбһўбһ¶бһ…бһ‘бҹҶбһ“бҹ„бһҡбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бҹ” бһ—бһ¶бһ–бһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҮбһ¶бһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ“бҹғбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”
  - бһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһҮбһ¶бһҮбһ¶бһ“бҹӢбһҹбҹ’бһ“бһјбһӣбһ“бҹғбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбҹ”
  - бһҳбһ“бһ»бһҹбҹ’бһҹбһӮбһәбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҗбһ¶бһҸбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһҫбһҖ/бһ”бһ·бһ‘ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ„бҹ’бһ бһ¶бһүбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһ–бҹҒбһӣбҹ”

## бһҳбһ¶бһӮбҹҢбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ

бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһҡбһ…бһ“бһ¶бһҖбһ“бҹ’бһӣбһ„бһҳбһҖ бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҳбһ¶бһӮбҹҢбһ¶бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. **бһ—бһ¶бһ–бһ”бһҫбһ—бҹ’бһӣбһә**бҹ– бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһҠбһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҗбһ¶ AI бһ”бһ¶бһ“бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһҡбһ”бҹҖбһ”бһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ (бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҖбһ“бҹ’бһӣбһ„бһҳбһҖ) бһ бһҫбһҷбһҡбһ”бҹҖбһ”бһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ бһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”
2. **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„**бҹ– бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҖбҹӮбһҒбҹ’бһӣбһҪбһ“ бһҸбҹӮбһ„бһҸбһ¶бһҳбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ бһҫбһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһӣбһҖбҹ’бһҒбһҺбҹҲбһҡбһ”бһҹбҹӢбһңбһ¶ (бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӣбһ”бҹӢбһ”бһ¶бһҸбҹӢ)бҹ”
3. **бһ—бһ¶бһ–бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶**бҹ– бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҠбҹӮбһӣбһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ–бһ бһ»бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһ“бһ¶бһ“бһ¶бҹ” бһ”бҹ’бһҡбһҫбһ’бһ¶бһҸбһ» UI/UX бһҠбҹӮбһӣбһҹбҹ’бһӮбһ¶бһӣбҹӢбһӮбҹ’бһ“бһ¶ бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ‘бҹ…бһ”бһ¶бһ“ (бһ§. бһҡбһјбһ”бһҹбһүбҹ’бһүбһ¶бһӮбҹ’бһҡбһ¶бһ”бҹӢбһҹбһҳбҹ’бһӣбҹҒбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҠбҹ„бһҷбһҹбһҳбҹ’бһӣбҹҒбһ„) бһ бһҫбһҷбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһҡбҹҲбһ…бһ·бһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбһҸбһ·бһҗбһ·бһҮбһ“бһІбҹ’бһҷбһ”бһ¶бһ“бһ…бҹ’бһҡбһҫбһ“бһ”бҹҶбһ•бһ»бһҸ (бһ§. бһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҒбҹ’бһӣбһёбҹ— бһҳбһ¶бһ“бһҮбҹҶбһ“бһҪбһҷбһңбһёбһҹбҹ’бһңбһӣ бһ“бһ·бһ„бһҳбһ¶бһҸбһ·бһҖбһ¶бһҡ "бһҡбҹҖбһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ")бҹ”

## бһҡбһ”бҹҖбһ”бһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸ бһ“бһ·бһ„бһҳбһ¶бһӮбҹҢбһ¶бһҒбһ¶бһ„бһ“бҹҒбҹҮ

бһҹбҹ’бһҡбһҳбҹғбһҗбһ¶бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҡбһ…бһ“бһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡ (Travel Agent), бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһӮбһ·бһҸбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҸбһ“бһ·бһ„бһҳбһ¶бһӮбҹҢбһ¶бҹ–

1. **бһ—бһ¶бһ–бһ”бһҫбһ—бҹ’бһӣбһә** вҖ“ бһўбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҠбһ№бһ„бһҗбһ¶ Travel Agent бһӮбһәбһҮбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣбһҳбһ¶бһ“ AI бһҮбһҪбһҷбҹ” бһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ–бһёбһҡбһ”бҹҖбһ”бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ (бһ§. бһҹбһ¶бһҡ "бһҹбһҪбһҹбҹ’бһҸбһё", бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ”бҹ’бһҡбһҳбһ¶бһҺбһ¬бһҖбһ¶бһҡбһҹбҹ’бһ“бһҫбһҹбһ»бҹҶ)бҹ” бһҹбһҡбһҹбҹҒбһҡвҖӢбһўбһҸбҹ’бһҗбһ”бһ‘бһ“бҹҒбҹҮбһІбҹ’бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһҫбһ‘бҹҶбһ–бҹҗбһҡбһ•бһӣбһ·бһҸбһ•бһӣбҹ” бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһүбҹ’бһҮбһёбһ“бҹғбһҹбҹҶбһҺбһҫбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбҹ’бһ“бһҫбһҹбһ»бҹҶбһҖбҹ’бһ“бһ»бһ„бһўбһҸбһёбһҸбһҖбһ¶бһӣбҹ” бһ”бһ„бҹ’бһ бһ¶бһүбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҡбһ”бҹҖбһ”бһ•бҹ’бһҸбһӣбҹӢбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢ (бһ…бһ»бһ…бһўбһ„бҹ’бһҡбһ»бһҸбһЎбһҫбһ„ бһ“бһ·бһ„бһ…бһ»бһ…бһўбһ„бҹ’бһҡбһ»бһҸбһ…бһ»бҹҮ, бһ”бҹҠбһјбһҸбһ»бһ„ Send Feedback, бһӣбҹ”) бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ¬бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһҺбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһҖбҹҶбһҺбһҸбҹӢбҹ”
2. **бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„** вҖ“ бһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһўбһ¶бһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҡбһҝбһ„бҹ—бһҠбһјбһ…бһҮбһ¶ System Promptбҹ” бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҗбһ¶бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ“бһ·бһҷбһ¶бһҷбһ…бҹ’бһҡбһҫбһ“бһ”бҹүбһ»бһҺбҹ’бһҺбһ¶ бһҳбһ¶бһ“бһҡбһ…бһ“бһ¶бһ”бһ‘бһҹбһҡбһҹбҹҒбһҡбһҷбҹүбһ¶бһ„бһҠбһјбһ…бһҳбҹ’бһҸбҹҒбһ… бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһ–бҹ’бһҡбһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһҠбҹӮбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһ·бһ“бһӮбһҪбһҡбһ“бһ·бһҷбһ¶бһҷбһўбҹҶбһ–бһёбҹ” бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһҫбһӣ бһ“бһ·бһ„бһӣбһ»бһ”бһҜбһҖбһҹбһ¶бһҡ бһ¬бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’ бһҖбҹҶбһҺбһҸбҹӢбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһҖбһ“бҹ’бһӣбһ„бһҳбһҖбҹ”
3. **бһ—бһ¶бһ–бһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶** вҖ“ бһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһІбҹ’бһҷа№Ғаёҷбһҡбһјбһ”бһ—бһ¶бһ–бһҹбһүбҹ’бһүбһ¶бһҖбҹ’бһ“бһ»бһ„бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҹбҹҶбһҺбһҫ (Share Prompt), бһ”бһ“бҹ’бһҗбҹӮбһҳбһҜбһҖбһҹбһ¶бһҡбһ¬бһҡбһјбһ”бһҗбһҸ бһ“бһ·бһ„бһҸбһ¶бһҖбҹӢбһҹбҹ’бһӣбһ¶бһҖбһ“бһҡбһҺбһ¶бһҳбҹ’бһ“бһ¶бһҖбҹӢ бһ¬бһўбҹ’бһңбһёбһҳбһҪбһҷ бһҳбһ¶бһ“бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶ бһ“бһ·бһ„бһўбһ¶бһ…бһҹбҹ’бһӮбһ¶бһӣбҹӢбһ”бһ¶бһ“бҹ” бһ”бҹ’бһҡбһҫбһҡбһјбһ”бһҹбһүбҹ’бһүбһ¶бһҖбҹ’бһҡбһңбһ¶бһҸбҹӢбһҖбҹ’бһҡбһҠбһ¶бһҹ (paperclip) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһ….upload/бһ…бҹӮбһҖбһҡбҹҶбһӣбҹӮбһҖбһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ бһ“бһ·бһ„бһҡбһјбһ”бһҹбһүбҹ’бһүбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһ¶бһҡбһўбһ¶бһ”бҹӢбһЎбһјбһҸбһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһҖбҹ”
