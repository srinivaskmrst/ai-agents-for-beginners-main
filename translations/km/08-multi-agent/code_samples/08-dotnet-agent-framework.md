## វាអ្វីដែលឧទាហរណ៍នេះបង្ហាញ

ប្រព័ន្ធដំណើរការ​ពហុភ្នាក់ងារនេះ បង្កើតសេវាកម្មណែនាំការធ្វើដំណើរនៅសណ្ឋាគារ ជាមួយភ្នាក់ងារពិសេសពីរ៖

1. **FrontDesk Agent**: ភ្នាក់ងារធ្វើដំណើរដែលផ្តល់ណែនាំអំពីសកម្មភាព និងទីតាំង
2. **Concierge Agent**: ពិនិត្យណែនាំ ដើម្បីធានាថាបទពិសោធន៍គឺពិត និងមិនមែនជាកន្លែងទេសចរណ៍ពេញនិយម

ភ្នាក់ងារទាំងនេះធ្វើការរួមគ្នាក្នុងលំហីតដែល៖
- ភ្នាក់ងារ FrontDesk ទទួលសំណើធ្វើដំណើរដំបូង
- ភ្នាក់ងារ Concierge ពិនិត្យនិងកែលម្អការណែនាំ
- លំហីតផ្តល់ចរន្តចម្លើយនៅពេលពិតប្រាកដ

## គំនិតសំខាន់

### Agent Coordination
ឧទាហរណ៍នេះបង្ហាញពីការសម្របសម្រួលភ្នាក់ងារដែលមានប្រភេទប្រកបដោយសុវត្ថិភាព ដោយប្រើ Microsoft Agent Framework ជាមួយការត្រួតពិនិត្យនៅពេលកំណត់។

### Workflow Orchestration
ប្រើការកំណត់ដំណើរការយ៉ាងសេចក្តីថ្លែង (declarative) ជាមួយលំនាំ async របស់ .NET ដើម្បីភ្ជាប់ភ្នាក់ងារច្រើនក្នុងបំពង់មួយ។

### Streaming Responses
អនុវត្តចរន្តពេលពិតនៃចម្លើយភ្នាក់ងារ ដោយប្រើ async enumerables និងស្ថាបត្យកម្មដឹកនាំដោយព្រឹត្តិការណ៍។

### Enterprise Integration
បង្ហាញលំនាំរួចរាល់សម្រាប់ផលិតកម្ម រួមមាន:
- ការកំណត់អថេរបរិស្ថាន
- ការគ្រប់គ្រងឯកសារបញ្ជាក់ឆ្លងកាត់យ៉ាងសុវត្ថិភាព
- ការដោះស្រាយកំហុស
- ការដំណើរការព្រឹត្តិការណ៍អស៊ីនក្រូន
