## рҹҺҜ бһңбһҸбҹ’бһҗбһ»бһ”бҹҶбһҺбһ„бһҡбҹҖбһ“

### рҹҡҖ **бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ**
- **Parallel Agent Execution**: бһҡбһҸбҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ AI бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһўбһҸбһ·бһ”бһҡбһҳбһ¶
- **Async/Await Patterns**: бһ”бҹ’бһҡбһҫбһӮбҹҶбһҡбһјбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё async бһҡбһ”бһҹбҹӢ .NET бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **GitHub Models Integration**: бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһ бҹ…бһҳбҹүбһјбһҢбҹӮбһӣ AI бһҡбһ”бһҹбҹӢ GitHub бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ
- **Resource Management**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ’бһ“бһ’бһ¶бһ“бһҳбҹүбһјбһҢбҹӮбһӣ AI бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ

### рҹҸ—пёҸ **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ**
- **Task-Based Parallelism**: бһ”бҹ’бһҡбһҫ .NET Task Parallel Library бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһҠбҹӮбһӣбһўoptбһёбһҳбҹүбһ¶бһӣбҹӢ
- **Synchronization Patterns**: бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһҒбһҺбҹҲбһҠбҹӮбһӣбһҮбҹҖбһҹбһңбһ¶бһ„бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ”бҹ’бһҡбһҺбһ№бһ„бһ‘бһӣбҹӢ
- **Load Balancing**: бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбҹҶбһ“бһҪбһүбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбһ¶бһҡбһҖбҹӮбһң
- **Fault Tolerance**: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ—бһ¶бһ–бһҒбһјбһ…бһҒбһ¶бһҸбһ“бҹ…бһӣбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ„бҹ’бһҒбҹҶбһұбҹ’бһҷбһӣбҹҶбһ бһ¶бһҸбҹӢбһҹбһҡбһ»бһ”бһҲбһ”бҹӢ

### рҹҸў **бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбҹҒбһ“бһ’бһҫбһ”бҹ’бһҡбҹүбһ¶бһҷмҰҲбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ**
- **High-Volume Document Processing**: бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҜбһҖбһҹбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ
- **Real-Time Content Analysis**: бһңбһ·бһ—бһ¶бһӮбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ…бһјбһӣбһҳбһҖбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһ–бһ·бһҸбһҮбһ¶бһҹбһҳбҹҗбһҷ
- **Batch Processing Optimization**: бһ”бһ„бҹ’бһҖбһҫбһ“бһ•бһӣбһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹҶбһ бҹҶбһ’бҹҶ
- **Multi-Modal Analysis**: бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҳбһ¶бһҸбһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһҳбҹҗбһҷбһ–бһёбһҡбһ¬бһ…бҹ’бһҡбһҫбһ“

- **Agent Pool**: бһўбҹ’бһ“бһҖбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ
- **Result Aggregation**: бһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ
- **Synchronization Points**: бһ”бҹ’бһҡбһ¶бһҖбһҠбһ—бһ¶бһ–бһ“бҹғбһҸбһҳбҹ’бһӣбһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ

## рҹҸў **бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹҒбһ“бһ’бһҫбһ”бҹ’бһҡбҹүбһ¶бһҷмҰҲ**

### вҡЎ **Throughput & Scalability**
- **Linear Performance Scaling**: бһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ•бһӣбһ…бҹҒбһү
- **Resource Utilization**: бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбҹүбһјбһҢбҹӮбһӣ AI бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–
- **Reduced Processing Time**: бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһ”бһ“бҹ’бһҸбһҸбһ¶бһҳбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ–бһ бһ»бһҹбһҳбҹҗбһҷ
- **Elastic Scaling**: бһҠбһ¶бһҖбҹӢбһ”бһҡбһ·бһҳбһ¶бһҺбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһҷбҹүбһ¶бһ„бһўбһ¶бһ…бһ”бһҸбҹӢбһ”бҹӮбһ“бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡ

### рҹӣЎпёҸ **Reliability & Resilience**
- **Fault Isolation**: бһҖбһ¶бһҡбһҒбһјбһ…бһҒбһ¶бһҸбһ“бҹ…бһӣбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбҹӮбһҳбһҪбһҷбһҳбһ·бһ“бһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҠбһӣбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ
- **Graceful Degradation**: бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹ…бһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҖбһҳбҹ’бһҡбһ·бһҸбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҸбһ·бһ…бһҮбһ¶бһ„бһҳбһ»бһ“
- **Error Recovery**: бһҳбҹүбҹҒбһҖбһ¶бһ“бһ·бһҹбҹҠбһёбһҳ Retry бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһҠбҹӮбһӣбһ”бһҡбһ¶бһҮбҹҗбһҷ
- **Load Distribution**: бһ…бҹӮбһҖбһ…бһ¶бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҷбҹүбһ¶бһ„бһҹбҹ’бһҳбһҫбһҹбҹ’бһҳбһ¶бһҸбһ“бҹ…бһӣбһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“

### рҹ“Ҡ **бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–**
- **Concurrent Execution Metrics**: бһҸбһ¶бһҳбһҠбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ“бҹғбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ–бһ бһ»бһҹбһҳбҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
- **Resource Usage Analytics**: бһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ CPU, бһҳбҹүбҹҒбһҳбһјбһҡбһё бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ¶бһү
- **Throughput Analysis**: бһңбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһҖбһҫбһ“бһЎбһҫбһ„бһ“бҹғбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ
- **Bottleneck Detection**: бһҖбҹҶбһҺбһҸбҹӢ бһ“бһ·бһ„бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһҺбһҸбҹӢбһҒбҹ’бһҹбҹ„бһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–

### рҹ”§ **бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ**
- **Async Programming Model**: бһ”бҹ’бһҡбһҫбһӮбҹҶбһҡбһј async/await бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбҹ’бһ”бһёбһ“бҹ… .NET
- **Task Coordination**: бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡ
- **Exception Handling**: бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹбһҠбҹҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ
- **Debugging Support**: бһ§бһ”бһҖбһҡбһҺбҹҚ Debugging бһҡбһ”бһҹбҹӢ Visual Studio бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӣбҹҶбһ бһ¶бһҸбҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ

бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҶбһ бһ¶бһҸбҹӢ AI бһҮбһ¶бһҹбһҳбҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҮбһ¶бһҳбһҪбһҷ .NET! рҹҡҖ

## рҹ’» бһҖбһ¶бһҡбһҡбһҸбҹӢбһҖбһјбһҠ

бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бҹҒбһүбһӣбҹҒбһүбһҳбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ `03.dotnet-agent-framework-workflow-ghmodel-concurrent.cs`бҹ” бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһё **Fan-Out/Fan-In concurrent workflow** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһүбһҠбҹҶбһҺбһҫбһҡ:

### рҹҸ—пёҸ **бһҹбҹ’бһҗбһ¶бһ”бһҸбҹ’бһҷбһҖбһҳбҹ’бһҳбһӣбҹҶбһ бһ¶бһҸбҹӢ**

```
User Request вҶ’ ConcurrentStartExecutor вҶ’ [Researcher Agent || Planner Agent] вҶ’ ConcurrentAggregationExecutor вҶ’ Final Output
```

**бһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—:**

1. **ConcurrentStartExecutor**: бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбҹҶбһҺбһҫбһҡбһ–бһёбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ
2. **Researcher Agent**: бһңбһ·бһ—бһ¶бһӮбһӮбҹ„бһӣбһҠбҹ… бһ“бһ·бһ„бһ‘бһ№бһ…бһӣбҹҶбһўбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһ¶бһҸбҹӢбһҮбһ¶бһҹбһҳбҹҗбһҷ
3. **Planner Agent**: бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҠбҹ„бһҷбһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһӣбҹҶбһ бһ¶бһҸбҹӢбһҮбһ¶бһҹбһҳбҹҗбһҷ
4. **ConcurrentAggregationExecutor**: бһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһ…бҹ’бһҡбһ”бһ¶бһ…бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ

### рҹҺҜ **бһӣбҹҶбһ“бһ¶бҹҶ Fan-Out/Fan-In**

бһӣбҹҶбһ бһ¶бһҸбҹӢбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһӣбҹҶбһ“бһ¶бҹҶбһ”бҹӮбһ”бһ”бһ»бһҡбһ¶бһҺ **Fan-Out/Fan-In**:
- **Fan-Out**: бһҹбһ¶бһҡ ОөО№Пғбҹ’бһҡбһҪбһ…бһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ…бҹ’бһҡбһҫбһ“бһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ
- **Concurrent Processing**: бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһӣбһҫбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡа№Җаё”аёөаёўаё§
- **Fan-In**: бһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһҡбһҪбһҳбһ…бҹҶбһҺбһјбһӣбһ‘бҹ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бҹҒбһүбһҸбҹӮбһҳбһҪбһҷ

### рҹ“қ бһӣбһ‘бҹ’бһ’бһ•бһӣбһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ

бһӣбҹҶбһ бһ¶бһҸбҹӢбһ“бһ№бһ„:
1. **Broadcast Request**: бһ•бҹ’бһүбһҫ "Plan a trip to Seattle in December" бһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡ
2. **Concurrent Processing**: бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ:
   - Researcher бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһ‘бһ¶бһҖбҹӢбһ‘бһ¶бһү бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ
   - Planner бһ”бһ„бҹ’бһҖбһҫбһҸ Itinerary бһ“бһ·бһ„бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“
3. **Aggregation**: бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһ‘бҹ…бһҮбһ¶бһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбһҡбһ»бһ”
4. **Display Results**: бһ”бһ„бҹ’бһ бһ¶бһүбһ•бҹӮбһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ…бҹ’бһҡбһ”бһ¶бһ…бҹӢбһҮбһ¶бһҳбһҪбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ

### рҹҺҜ бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠ

бһӣбҹҶбһ“бһ¶бҹҶбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷбһ“бҹҒбҹҮбһҹбһҳбһҹбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ:
- **Content Creation**: бһўбҹ’бһ“бһҖбһ“бһ·бһ–бһ“бҹ’бһ’бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹ…бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ
- **Code Review**: бһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ…бҹ’бһҡбһҫбһ“бһңбһ·бһ—бһ¶бһӮбһҖбһјбһҠбһ–бһёбһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—
- **Market Research**: бһңбһ·бһ—бһ¶бһӮбһ•бҹ’бһҹбһ¶бһҡбһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—
- **Document Processing**: бһҠбһҖбһҷбһҖ, бһңбһ·бһ—бһ¶бһӮ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹ„бһҷбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҮбһ¶бһҹбһҳбҹҗбһҷ
- **Multi-Perspective Analysis**: бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ‘бһҹбҹ’бһҹбһ“бһңбһ·бһҹбҹҗбһҷбһ…бҹ’бһҡбһҫбһ“бһӣбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбһјбһ…бһӮбҹ’бһ“бһ¶

### рҹ”Қ бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһё Executors бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

**ConcurrentStartExecutor:**
- Implements `IMessageHandler<string>` бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһўбһүбҹ’бһүбһ¶бһҺбһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһҹбһ¶бһҡбһҒбҹ’бһҹбҹӮбһўбһҖбҹ’бһҹбһҡ
- бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһҹбһ¶бһҡвҖӢбһ‘бҹ…бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡвҖӢбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ
- бһ•бҹ’бһүбһҫ `TurnToken` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҹбһҳбҹҗбһҷ

**ConcurrentAggregationExecutor:**
- Implements `IMessageHandler<ChatMessage>` бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ‘бһҪбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡ
- бһ”бҹ’бһҡбһҳбһјбһӣбһҹбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ…бҹ’бһҡбһҖбһ…бһҡ
- бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһ–бҹҒбһӣбһҠбҹӮбһӣбһ…бһҳбҹ’бһӣбһҫбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбҹҶбһ–бһ№бһ„бһҳбһҖбһҠбһӣбҹӢ
- бһ”бһ„бҹ’бһ бһ¶бһүбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `context.YieldOutputAsync()`

### вҡЎ бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–

**Concurrent vs Sequential:**
- Sequential: Agent1 (30s) вҶ’ Agent2 (30s) = **бһҹбһҡбһ»бһ” 60 бһңбһ·бһ“бһ¶бһ‘бһё**
- Concurrent: Agent1 (30s) || Agent2 (30s) = **бһҹбһҡбһ»бһ” 30 бһңбһ·бһ“бһ¶бһ‘бһё**

**Throughput improvement**: бһӣбһҝбһ“бһҠбһӣбҹӢ NГ— бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶ N (бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡ бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“)

### рҹӣЎпёҸ бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹ

бһӣбҹҶбһ бһ¶бһҸбҹӢбһ“бҹҒбҹҮбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбһҡбһҺбһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһ”бһҡбһ¶бһҮбҹҗбһҷбһҠбҹ„бһҷбһҷбҹүбһ¶бһ„бһ‘бһ“бҹӢбһ—бҹ’бһӣбһ“бҹӢ:
- бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҳбһҪбһҷбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһ“бҹ…бһҸбҹӮбһ”бһ“бҹ’бһҸбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ
- Aggregator бһўбһ¶бһ…бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бҹҒбһӣбһҖбҹҶбһҺбһҸбҹӢ (timeout)
- бһўбһ¶бһ…бһҸбҹ’бһҡбһЎбһ”бҹӢбһӣбһ‘бҹ’бһ’бһ•бһӣбһҒбҹ’бһӣбҹҮбҹ—бһ”бһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢ

### рҹ“Ҡ бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
