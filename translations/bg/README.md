> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Това ви дава всичко необходимо за завършване на курса с много по-бързо изтегляне.

## 🌱 Започване

Този курс съдържа уроци, покриващи основите на изграждането на AI агенти. Всеки урок разглежда своя тема, така че започнете откъдето пожелаете!

Има поддръжка на множество езици за този курс. Вижте нашите [налични езици тук](#-multi-language-support).

Ако е първият път, в който работите с генеративни AI модели, разгледайте нашия курс [Generative AI For Beginners](https://aka.ms/genai-beginners), който включва 21 урока за работа с GenAI.

Не забравяйте да [звездичкате (🌟) това хранилище](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) и да [форкнете това хранилище](https://github.com/microsoft/ai-agents-for-beginners/fork), за да стартирате кода.

### Срещнете други обучаващи се, получете отговори на вашите въпроси

Ако имате затруднения или въпроси относно изграждането на AI агенти, присъединете се към нашия специализиран канал в Discord в [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Какво ви е необходимо

Всеки урок в този курс включва примери с код, които могат да бъдат намерени в папката code_samples. Можете да [форкнете това хранилище](https://github.com/microsoft/ai-agents-for-beginners/fork), за да създадете свое собствено копие.

Примерите с код в тези упражнения използват Microsoft Agent Framework с Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Изисква Azure акаунт

Този курс използва следните AI Agent фреймуърци и услуги от Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Някои примери с код също поддържат алтернативни доставчици, съвместими с OpenAI, като например [MiniMax](https://platform.minimaxi.com/), който предлага модели с голям контекст (до 204 хиляди токена). Вижте [Course Setup](./00-course-setup/README.md) за подробности по конфигурацията.

За повече информация относно стартирането на кода за този курс, отидете на [Course Setup](./00-course-setup/README.md).
