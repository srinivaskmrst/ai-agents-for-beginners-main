## 🌱 Začíname

Tento kurz obsahuje lekcie pokrývajúce základy tvorby AI Agentov. Každá lekcia sa venuje svojej téme, takže začnite tam, kde chcete!

Kurz má podporu viacerých jazykov. Nájdete ju v našom [dostupných jazykoch tu](#-multi-language-support). 

Ak je toto váš prvýkrát, čo tvoríte s modelmi Generatívnej AI, pozrite si náš kurz [Generative AI For Beginners](https://aka.ms/genai-beginners), ktorý obsahuje 21 lekcií o tvorbe s GenAI.

Nezabudnite [dát hviezdičku (🌟) tomuto repozitáru](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [forknúť tento repozitár](https://github.com/microsoft/ai-agents-for-beginners/fork), aby ste mohli spustiť kód.

### Spoznajte ostatných študentov, získajte odpovede na svoje otázky

Ak sa zaseknete alebo máte akékoľvek otázky o tvorbe AI Agentov, pripojte sa na náš vyhradený Discord kanál v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Čo potrebujete

Každá lekcia v tomto kurze obsahuje príklady kódu, ktoré nájdete v priečinku code_samples. Môžete si [forknúť tento repozitár](https://github.com/microsoft/ai-agents-for-beginners/fork) a vytvoriť si vlastnú kópiu.  

Príklady kódu v týchto cvičeniach využívajú Microsoft Agent Framework spolu s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Vyžaduje sa Azure účet

Tento kurz využíva nasledujúce AI Agent frameworky a služby od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Niektoré príklady kódu podporujú aj alternatívnych poskytovateľov kompatibilných s OpenAI, ako napríklad [MiniMax](https://platform.minimaxi.com/), ktorý ponúka modely s veľkým kontextom (až do 204K tokenov). Pozrite si [Nastavenie kurzu](./00-course-setup/README.md) pre podrobnosti konfigurácie.

Pre viac informácií o spúšťaní kódu tohto kurzu navštívte [Nastavenie kurzu](./00-course-setup/README.md).
