## 🌱 စတင်ထိုးတက်ခြင်း

ဒီသင်တန်းမှာ AI အေးဂျင့်တွေ တည်ဆောက်ရာ ရည်ရွယ်ချက်နဲ့ အခြေခံအကြောင်းအရာတွေ ပါရှိပါတယ်။ အချိန်မရွေး ကိုယ်ကြိုက်ရာ ကနေ စတင်လေ့လာနိုင်ပါတယ်!

ဒီသင်တန်းမှာ ဘာသာစကားများစွာ ထောက်ပံ့ထားပါတယ်။ ကျွန်တော်တို့ရဲ့ [နောက်ထပ်ဘာသာစကားများထောက်ပံ့ချက်](#-multi-language-support) ကို သွားကြည့်ပါ။

Generative AI မော်ဒယ်တွေနဲ့ ပထမဆုံးအကြိမ် အလုပ်လုပ်ဖို့ စဉ်းစားနေရင် [Generative AI For Beginners](https://aka.ms/genai-beginners) သင်တန်းကိုလည်း မမေ့ပဲ ကြည့်ပါ။ ၎င်းမှာ GenAI ကို အသုံးပြုပြီး တည်ဆောက်မည့် ၂၁ ခုသော သင်ခန်းစာများ ပါပါတယ်။

ဒီ repo ကို [ကြယ်စင် (🌟) ပေးခြင်း](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) နှင့် ကုဒ် ကို ပြေးနိုင်ဖို့ [fork လုပ်ခြင်း](https://github.com/microsoft/ai-agents-for-beginners/fork) မမေ့နဲ့နော်။

### အခြားသင်တန်းသားတွေနဲ့ တွေ့ဆုံ၊ မေးခွန်းတွေ ဖြေကြားပါ

AI အေးဂျင့်တည်ဆောက်ရာမှာ ပြဿနာဖြစ် ဆုတ်သွားရင် သို့မဟုတ် မေးခွန်းရှိရင် Microsoft Foundry Discord ထဲက [Discord ချန်နယ်](https://aka.ms/ai-agents/discord) မှာ ဝင်ရောက်ဆွေးနွေးကြပါ။

### လိုအပ်တာတွေ

ဒီသင်တန်းမှာ တစ်ခုချင်းစီ သင်ခန်းစာတွေအတွက် တစ်ခြား့နမူနာကုဒ်များ ပါဝင်ပြီး code_samples ဖိုလ်ဒါထဲမှာ တွေ့နိုင်ပါတယ်။ ကိုယ်ပိုင် ရပ်တည်ချက်ဖန်တီးဖို့ ဒီ repo ကို [fork လုပ်ပါ](https://github.com/microsoft/ai-agents-for-beginners/fork)။

ဒီလေ့ကျင့်ခန်းတွေမှာ Microsoft Agent Framework နဲ့ Azure AI Foundry Agent Service V2 ကို အသုံးပြုထားပါတယ် -

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure အကောင့် လိုအပ်ပါသည်

ဒီသင်တန်းသည် Microsoft မှ ရရှိနိုင်သော AI Agent frameworks နှင့် ဝန်ဆောင်မှုများကို အသုံးပြုသည် -

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

အချို့သော ကုဒ်နမူနာများတွင် OpenAI နှင့် ဆင်တူ MiniMax (ထိပ်တန်း တိုးရှင် ၂၀၄,၀၀၀-token မော်ဒယ်များပါဝင်) ကဲ့သို့သော ပံ့ပိုးသူများကိုလည်း အသုံးပြုနိုင်သည်။ ဖော်ပြချက်အတွက် [Course Setup](./00-course-setup/README.md) ကို ကြည့်ပါ။

ဒီသင်တန်းရဲ့ ကုဒ်များ မည်သို့ ပြေးရမည်ဆိုတာ အကြောင်းကို [Course Setup](./00-course-setup/README.md) မှာ ပိုမို ရယူနိုင်ပါသည်။
