> **स्थानीय रूप से क्लोन करना पसंद करते हैं?**
>
> यह रिपॉजिटरी 50+ भाषा अनुवाद शामिल करता है जो डाउनलोड साइज़ को काफी बढ़ा देता है। अनुवाद बिना क्लोन करने के लिए, sparse checkout का उपयोग करें:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> इससे आपको बहुत तेज़ डाउनलोड के साथ कोर्स पूरा करने के लिए आवश्यक सब कुछ मिल जाता है।

## 🌱 शुरुआत कैसे करें

यह कोर्स AI एजेंट बनाने के मूलभूत सिद्धांतों को कवर करता है। हर पाठ अपने विषय को कवर करता है, इसलिए जहाँ चाहें वहीं से शुरू करें!

इस कोर्स के लिए बहुभाषी समर्थन है। हमारी [यहाँ उपलब्ध भाषाओं को देखें](#-multi-language-support)। 

यदि यह आपकी पहली बार जीनेरेटिव AI मॉडल के साथ निर्माण है, तो हमारा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स देखें, जिसमें GenAI के साथ निर्माण पर 21 पाठ शामिल हैं।

कोड चलाने और इस रिपॉजिटरी को उपयोग करने के लिए इसे [स्टार (🌟) करें](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) और [फोर्क करें](https://github.com/microsoft/ai-agents-for-beginners/fork)।

### अन्य शिक्षार्थियों से मिलें, अपने प्रश्नों के उत्तर पाएं

अगर आप अटक जाते हैं या AI एजेंट बनाने को लेकर कोई भी प्रश्न है, तो हमारे समर्पित Discord चैनल में शामिल हों [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)।

### आपको क्या चाहिए

इस कोर्स के प्रत्येक पाठ में कोड उदाहरण शामिल हैं, जिन्हें code_samples फोल्डर में पाया जा सकता है। आप अपनी प्रति बनाने के लिए [रिपॉजिटरी फोर्क कर सकते हैं](https://github.com/microsoft/ai-agents-for-beginners/fork)।

इन अभ्यासों में कोड उदाहरण Microsoft Agent Framework और Azure AI Foundry Agent Service V2 का उपयोग करते हैं:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाता आवश्यक

यह कोर्स Microsoft के निम्न AI एजेंट फ्रेमवर्क और सेवाओं का उपयोग करता है:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

कुछ कोड उदाहरण वैकल्पिक OpenAI-अनुकूल प्रदाताओं का भी समर्थन करते हैं जैसे [MiniMax](https://platform.minimaxi.com/), जो बड़े संदर्भ मॉडल (204K टोकन तक) प्रदान करता है। कॉन्फ़िगरेशन विवरण के लिए [Course Setup](./00-course-setup/README.md) देखें।

इस कोर्स का कोड चलाने के बारे में अधिक जानकारी के लिए, [Course Setup](./00-course-setup/README.md) देखें।
