# கணினி பயன்பாட்டு முகவர்கள் (CUA) உருவாக்கல்

கணினி பயன்பாட்டு முகவர்கள் ஒரு நபர் செய்வது போலவே வலைத்தளங்களுடன் தொடர்பு கொள்ள முடியும்: ஒரு உலாவியை திறந்து, பக்கத்தை ஆய்வு செய்து, அதை காணும் மற்றும் அடுத்த சிறந்த நடவடிக்கையை எடுத்துச் செயல்படுவது. இந்த பாடத்தில், நீங்கள் ஒரு உலாவி தானியங்கி முகவரியை உருவாக்கப்போகிறீர்கள், இது Airbnb ஐ தேடி, கட்டமைக்கப்பட்ட பட்டியல்கள் தரவை எடுத்து, ஸ்டாக்ஹோல்ம் நகரில் மிகவும் குறைந்த விலை இடத்தை கண்டறியும்.

இந்த பாடம் Browser-Use ஐ AI இயக்க நவிகேஷனுக்காக, Playwright மற்றும் Chrome DevTools படProtocol (CDP) ஐ உலாவி கட்டுப்பாட்டுக்கு, Azure OpenAI ஐ பார்வை சாணுணர்வுக்கு, மற்றும் Pydantic ஐ கட்டமைக்கப்பட்ட தரவீடு சுருக்கத்துக்கு சேர்க்கிறது.

## அறிமுகம்

இந்த பாடம் பின்வருவனைக் கையாளும்:

- API மட்டமான தானியக்கத்தைவிட கணினி பயன்பாட்டு முகவர்கள் எப்போது சிறந்தது என்பதை புரிந்துகொள்ளுதல்
- நம்பகமான உலாவி வாழ்க்கைச்சுழற்சி மேலாண்மைக்கு Browser-Use ஐ Playwright மற்றும் CDP உடன் ஒருங்கிணைப்பது
- மாற்றுத் தன்மை கொண்ட வலைப்பக்கங்களில் பட்டியல் தரவுகளை எடுக்க Azure OpenAI பார்வை மற்றும் கட்டமைக்கப்பட்ட Pydantic வெளியீட்டை பயன்படுத்தல்
- முகவர் முன், இயக்குனர் முன் அல்லது கலந்த வகை உலாவி தானியக்க வேலைவாயில்களை எப்போது பயன்படுத்த வேண்டும் என்பதைக் கணக்கிடுதல்

## கற்றல் குறிக்கோள்கள்

இந்த பாடத்தை முடித்த பிறகு, நீங்கள் எப்படி செய்ய வேண்டியதென்றால்:

- Azure OpenAI மற்றும் Playwright உடன் Browser-Use ஐ அமைத்தல்
- உண்மை வலைத்தளத்தில் பயணிக்கும் மற்றும் மாற்றுத் தன்மை IU கூறுகளை கையாளும் உலாவி தானியக்க வேலைவாயிலை உருவாக்கல்
- காட்சிக்கும் பக்க உள்ளடக்கம் என்பதில் இருந்து தட்டிய வகையை எடுத்துக் கொண்டு, அதனைக் காரியவாயிலும் தொழில்துறை விதிமுறைகளிலும் மாறுதல் செய்வது
- உலாவி பணிக்கு எவ்வாறு முன்னறிவிக்கப்பட்டது என்ற அடிப்படையில் முகவர் மற்றும் இயக்குனர் வடிவங்களை தேர்வு செய்தல்

## குறியீடு மாதிரி

இந்த பாடத்தில் ஒரு குறியீடு நோட்புக் குறியீடு உள்ளது:

- [15-browser-user.ipynb](./15-browser-user.ipynb): CDP ஓரத்தில் Chrome அமர்வை தொடங்குகிறது, ஸ்டாக்ஹோல்ம் பட்டியல்களுக்காக Airbnb தேடுகிறது, Browser-Use பார்வையால் விலைகளை எடுத்து, கட்டமைக்கப்பட்ட தரவாக மிகக் குறைந்த விலை வாய்ப்பைத் திருப்புகிறது.

## தேவையான முன்னோட்டங்கள்

- Python 3.12+
- உங்கள் சூழலில் Azure OpenAI அமர்வு அமைப்பு
- உள்ளகமாக Chrome அல்லது Chromium நிறுவல்
- Playwright சார்ந்தல் நிறுவல்
- async Python அடிப்படை பரிச்சயம்

## அமைப்பு

நோட்புக்கில் பயன்படுத்தப்படும் தொகுதிகளை நிறுவுக:

```bash
pip install browser_use playwright python-dotenv
playwright install chromium
```

நோட்புக் பயன்படும் Azure OpenAI சூழல் மாறிலிகளை அமைக்கவும்:

```bash
AZURE_OPENAI_ENDPOINT=...
AZURE_OPENAI_API_KEY=...
AZURE_OPENAI_CHAT_DEPLOYMENT_NAME=...
# விருப்பம்: தவிர்க்கப்பட்டால் தற்போதைய API பதிப்புக்கு இயல்பாக அமைக்கப்படும்
AZURE_OPENAI_API_VERSION=...
```

## கட்டமைப்பு கண்ணோட்டம்

நோட்புக் ஒரு கலந்த வகை உலாவி தானியக்க வேலைவாயிலை காட்டுகிறது:

1. Chrome CDP இயக்கத்துடன் துவங்கி, Playwright மற்றும் Browser-Use இரண்டுக்கும் ஒரே உலாவி அமர்வை பகிர முடியும்.
2. ஒரு Browser-Use முகவர், Airbnb ஓப்பனிங், பாப்-அப் நீக்குதல், ஸ்டாக்ஹோல்ம் தேடல் போன்ற திறன்பட்ட பயணத் பணிகளை கையாள்கிறது.
3. செயல்பாட்டில் உள்ள பக்கம் கட்டமைக்கப்பட்ட Pydantic திட்டத்தால் ஆய்வு செய்யப்படுகின்றது, பட்டியல் தலைப்புகள், இரவு விலை, மதிப்பீடுகள் மற்றும் URL-களை எடுக்க.
4. Python யுக்தி எடுத்த தரவுகளை ஒப்பிட்டு மிகக் குறைந்த விலை முன்னிலை வைக்கிறது.

இந்த அணுகுமுறை Browser-Use வழங்கும் მოქடும் பார்வை அடிப்படையுடைய சாணுணர்வையும், தேவைப்படும் போது திடமான உலாவி கட்டுப்பாட்டையும் வழங்குகிறது.

## முக்கியக் குறிப்புகள் மற்றும் சிறந்த நடைமுறைகள்

### முகவர் மற்றும் இயக்குனர் எப்போது பயன்படுத்துவது

| சூழல் | முகவரைக் கையாளவும் | இயக்குனரைப் பயன்படுத்தவும் |
|--------|-------------------|---------------------------|
| மாற்றம் அடைந்த படிவங்கள் | ஆம், AI பக்க மாற்றங்களுக்கு ஏற்ப ஒத்துழைக்கிறது | இல்லை, நன்றாக பாகுபடுத்தும் தேர்வுகளால் உடையும் |
| அறியப்பட்ட கட்டமைப்பு | இல்லை, முகவர் நேரடியான கட்டுப்பாட்டிலிருந்து மெதுவாக இருக்கிறது | ஆம், வேகமான மற்றும் துல்லியமானது |
| கூறுகள் தேடல் | ஆம், இயற்கை மொழி நன்கு செயல்படுகிறது | இல்லை, துல்லியத் தேர்வுகள் தேவை |
| நேர மேலாண்மை | இல்லை, குறைவான முன்னறிவிப்புடன் | ஆம், காத்திருக்கைகள் மற்றும் மறு முயற்சிகளுக்கு முழு கட்டுப்பாடு |
| சிக்கலான வேலைநிலைகள் | ஆம், எதிர்பாராத IU நிலைகளை கையாளும் | இல்லை, திறந்த கிளைகள் தேவை |

### Browser-Use சிறந்த நடைமுறைகள்

1. ஆய்வுக்கான மற்றும் தற்காலிக பயணத்திற்காக முகவர்கள் மூலம் துவங்கவும்.
2. தொடர்பு முன்னறிவிக்கப்பட்டபோது நேரடி பக்க கட்டுப்பாட்டுக்கு மாறவும்.
3. எடுக்கப்பட்ட தரவு சரிபார்க்கபடும் மற்றும் வகைப்பாட்டிற்கு பாதுகாப்பானதாக உள்ளது என்பதை உறுதி செய்ய கட்டமைக்கப்பட்ட வெளியீட்டு மாதிரிகளை பயன்படுத்தவும்.
4. காட்சியளிக்கும் IU மாற்றங்களுக்குப் பிறகு செயலில் தாமதங்களை நுணுக்கமாக சேர்க்கவும்.
5. தோல்விகள் எளிதில் குறைத்துக் கொள்ள புகைப்படங்களை எடுத்துக் கொள்ளவும்.
6. வலைத்தளங்கள் மாறக்கூடும் என்று எதிர்பார்த்து பாப்-அப்புகள் மற்றும் படிவ மாற்றங்களுக்கு மாற்று திட்டங்களை வடிவமைக்கவும்.
7. முகவர் மற்றும் இயக்குனர் வடிவங்களை கலக்கவும், அத்துடன் சுழற்சி மற்றும் துல்லியத்தை பெற்றுக் கொள்ளவும்.

### உண்மை உலக பயன்பாடுகள்

- பயண புக் செய்தல் மற்றும் விலை கண்காணிப்பு
- மின்னணுவணிக விலை ஒப்பீடு மற்றும் கிடைக்கும் நிலைகளை சோதனை
- மாற்றத்துடன் வலைத்தளங்களிலிருந்து கட்டமைக்கப்பட்ட எடுப்பு
- பார்வை அறிவு கொண்ட IU சோதனை மற்றும் உறுதிப்படுத்தல்
- வலைத்தள கண்காணிப்பு மற்றும் எச்சரிக்கை
- பல படி பணிகளில் அறிவார்ந்த படிவம் நிரப்புதல்
