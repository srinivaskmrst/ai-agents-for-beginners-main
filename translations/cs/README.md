## 🌱 Začínáme

Tento kurz obsahuje lekce pokrývající základy vytváření AI Agentů. Každá lekce pokrývá své vlastní téma, tak začněte, kde chcete!

Kurz má podporu více jazyků. Podívejte se na naše [dostupné jazyky zde](#-multi-language-support). 

Pokud stavíte poprvé s generativními AI modely, podívejte se na náš kurz [Generative AI For Beginners](https://aka.ms/genai-beginners), který obsahuje 21 lekcí o vytváření s GenAI.

Nezapomeňte [dát hvězdičku (🌟) tomuto repozitáři](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [rozdvojit (fork) tento repozitář](https://github.com/microsoft/ai-agents-for-beginners/fork), abyste mohli spustit kód.

### Poznejte další studenty, získejte odpovědi na své dotazy

Pokud se zaseknete nebo máte otázky ohledně vytváření AI Agentů, přidejte se na náš vyhrazený kanál Discord v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Co potřebujete

Každá lekce v tomto kurzu obsahuje ukázky kódu, které najdete ve složce code_samples. Můžete si [rozdvojit tento repozitář](https://github.com/microsoft/ai-agents-for-beginners/fork), abyste si vytvořili vlastní kopii.  

Ukázky kódu v těchto cvičeních využívají Microsoft Agent Framework s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Vyžaduje účet Azure

Tento kurz používá následující rámce a služby AI Agentů od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Některé ukázky kódu také podporují alternativní poskytovatele kompatibilní s OpenAI, například [MiniMax](https://platform.minimaxi.com/), který nabízí modely s dlouhým kontextem (až 204 tisíc tokenů). Podrobnosti o konfiguraci najdete v [Course Setup](./00-course-setup/README.md).

Pro více informací o spuštění kódu k tomuto kurzu přejděte do [Course Setup](./00-course-setup/README.md).
